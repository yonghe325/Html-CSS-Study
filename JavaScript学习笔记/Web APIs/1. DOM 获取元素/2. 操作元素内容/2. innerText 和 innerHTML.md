二者都为对象的一个属性
通过修改这些属性
可以更改对应标签的文字
# .innerText
```js
    <div class="box">我是文字内容</div>
    <script>
      //获取元素
       const box = document.querySelector(".box");
      //修改文字内容
      box.innerText = `<strong>我是修改之后的文字</strong>`
        //不会解析标签
      console.log(box.innerText);
    </script>
```
通过innerText属性修改文字
不会将修改后包含的标签所解析
会==原样修改==
# .innerHTML
```js
    <div class="box">我是文字内容</div>
    <script>
      //获取元素
      const box = document.querySelector(".box");
      box.innerHTML = `<strong>我是修改之后的文字</strong>`;
      //会解析标签
      console.log(box.innerHTML);
    </script>
```
将==标签解析==
