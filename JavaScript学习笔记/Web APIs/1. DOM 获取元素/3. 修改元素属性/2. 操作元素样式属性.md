# ==使用classList操作类控制css==
有三个基本操作
```js
对象.classList.add('类名')
//添加类
对象.classList.remove('类名')
//移除类
对象.classList.toggle('类名')
//切换类
	//类已有,就移除该类
	//类没有, 就添加该类
```
# 直接修改样式
对象.style.样式 = '值'
```js
const box = document.querySelector('.box')
box.style.width = '300px'
```
==值要包含单位==
当遇到通过 `-` 连接的样式时
将第二个单词的首字母大写, 并去掉 `-`
```js
box.style.backgroudColor = 'black'
box.style.borderTop = '2px soild black'
```
适用于对一个对象进行单独的修改
# 通过类名修改样式
对象.className= '类名'
```js
const box = document.querySelector('.box')
box.className = 'box2'
```
使用className修改类名
==会直接将原来的类名覆盖==
适用于修改样式比较多的情况
