
# 匹配一个元素,获得其匹配的对象
通过 ==querySelector== 方法获取==一个==对象
返回一个 ==对象==
```js
document.querySelector('[css选择器]')

    //获取第一个div标签
    const div =  document.querySelector('div')
    //获取class为box的标签
    const box =  document.querySelector('.box')
    //获取id为nav的标签
    const nav = document.querySelector('#nav')
    //获取ul中第一个li
    const li_1=document.querySelector('ul li:first-child')
```
可以直接对获取的对象修改其属性
# 获取多个对象
通过 ==querySelectorAll== 获取所有匹配的对象
返回 一个==对象**伪数组**==
```js
<body>
  <ul>
    <li>1</li>
    <li>2</li>
    <li>3</li>
    <li>4</li>
    <li>5</li>
    <li>6</li>
    <li>7</li>
  </ul>
  <script>
    const lis=document.querySelectorAll('ul li')
    console.log(lis)
  </script>
</body>
```
需要通过遍历获得需要的对象
在进行修改其属性
## 伪数组
有一般数组的下标,长度
单位有push pop等方法对其进行操作
想要得到里面的每一个对象
则需要遍历(for)的方式获得
```js
<body>
  <ul id="nav">
    <li>1</li>
    <li>2</li>
    <li>3</li>
  </ul>
  <script>
    const lis=document.querySelectorAll('#nav li')
    for (let i = 0; i < lis.length; i++) {
      console.log(lis[i])
    }
    // console.log(lis)
  </script>
</body>
```
