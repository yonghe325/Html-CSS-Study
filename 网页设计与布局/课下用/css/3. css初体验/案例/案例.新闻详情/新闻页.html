在希望的田野上 | 湖北秋收开镰 各地多举措保增产增收

来源 :央视网 | 2222年12月12日 12:12:12

央视网消息 : 眼下湖北秋收开镰已有一周有多的时间.水稻收割已超过四成,玉米收割七成.湖北省通过大力推广新品种水稻建设高标准农田等一系列措施,为秋粮稳产提供有力保障

中稻占据湖北省全年粮食产量的一半以上,在湖北生产区荆门市,370万亩中稻已经收割四成以上

王华林说的新品种,是湖北省研发的杂交水稻"华夏香丝",不仅产量高,还有扛到,抗病,抗高温的特性.在荆门市漳河镇的一工程示范田内,像"华夏香丝"这样的抗旱节水品种还有二十多个,这些水稻品种将在荆门市全面推广,确保来年增产增收

此外,湖北还大力推广高标准农田建设,截至今年六月,已建成3980万亩高标准农田.目前,湖北省仍有1800万亩中稻正在有序收割中,预计十月中旬收割完毕
