在HTML文件开头通过
`doctype` 标签指定文档模式
文档模式共有三种 : 
+ 混杂模式(quirks mode)
	直接省略 `doctype` 标签, 即使用混杂模式
+ 标准模式(standards mode)
	```js
	<!-- HTML5 -->
	<!DOCTYPE html>
	```
+ 准标准模式(almost standards mode )
	准标准模式通过*过渡性文档类型（Transitional）* 和 *框架集文档类型（Frameset）* 来触发

混杂模式 与 标准模式的区别主要在于 CSS渲染内容的差异
对JavaScript也有一些影响
准标准模式 与 标准模式 类似, 主要区别在于如何对待图片元素周围的空白
(在表格中使用图片最明显)
准标准模式与标准模式很接近, 一般不需要特别区分
