因为 JavaScript 为弱类型语言, 所以需要使用 `typeof` 来确定数据类型
 `typeof` 会根据输入的变量 返回以下字符串
 + "undefined" : 值未定义
 + "boolean" : 表示值为布尔型
+  "string" : 表示值为字符串；
+ "number" : 表示值为数值；
+ "symbol" : 表示值为符号 
+  "object" : 表示值为对象（而不是函数）或 null；
+  "function" : 表示值为函数；
```js
let message = "some string";
console.log(typeof message); // "string"
console.log(typeof(message)); // "string"
console.log(typeof 95); // "number"
```
特殊的是
当typeof Null 时, 返回的时object
原因是null被认为是一个空对象
