`Boolean`有两个字面值:
`true` 和 `false`
这两个布尔值不同于数值
所以`true`不等于1, `false`不等于0
# 布尔类型转换
使用`Boolean()`函数,可以将其他类型的值转换为`Boolean`类型的值

|   数据类型    |         true         |   false   |
| :-------: | :------------------: | :-------: |
|  Boolean  |         true         |   false   |
|  String   |        非空字符串         | ""(空字符串)  |
|  Number   |     为零数值(包括无限值)      |  0, NaN   |
|  Object   |         任意对象         |   Null    |
| Undefined | -------------------- | Undefined |
