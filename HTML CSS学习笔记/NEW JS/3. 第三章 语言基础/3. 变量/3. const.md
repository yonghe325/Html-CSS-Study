# const
+ 声明  不可重新赋值 的 ==块级作用域== 变量
	如果声明的是一个对象, 则它的属性可以被修改
+ 使用`const`声明的变量, 必须在声明时进行初始化
## const在for循环中
`const`在循环中与``let``相似
同样是块作用域, 同样是每次迭代都重新声明迭代变量
但不同的是
`const`无法`++`,所以const无法在普通的for循环中使用
但对于 for_in循环 和 for_of 循环来说
`const`每次都会重新声明, 重新绑定新的对象, 所以可以使用`const`来声明迭代变量
```js
let i = 0;
for (const j = 7; i < 5; ++i) {
	console.log(j);
}
// 7, 7, 7, 7, 7

for (const key in {a: 1, b: 2}) {
	console.log(key);
}
// a, b

for (const value of [1,2,3,4,5]) {
	console.log(value);
}
// 1, 2, 3, 4, 5
```
### 绑定
`const` 声明创建了 ==一个对值的不可变== 的==引用==
并不意味着它所持有的值是不可变的，只是==变量标识符==不能被重新赋值
即该变量不能再指向其他 引用类型
+ 对于使用``const``声明的对象/数组
	不能再 被覆盖 指向 其他 对象/数组
	但是其==键值对/元素==不会固定, 可以被修改/添加/删除
