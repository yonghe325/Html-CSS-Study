**RegExp（Regular Expression，正则表达式）** 
是 JavaScript 中用于**模式匹配**和**文本搜索**的内置对象类型
通过这种类型, 可以更方便的对字符串/数值等进行格式验证
```js
// 场景：验证邮箱格式
const email = "user@example.com";

// 1. 不用正则（麻烦且不灵活）
function checkEmailWithoutRegex(email) {
    return email.includes("@") && 
           email.includes(".") && 
           email.indexOf("@") < email.lastIndexOf(".");
}

// 2. 用正则（简单且强大）
const emailRegex = /^[\w-]+@[\w-]+\.[a-z]{2,}$/i;
const isValid = emailRegex.test(email);  // true
```
