# line-height -> 行高 
行高 = 上间距 + 文本高度 + 下间距
属性值 : 数字+单位(px) / 数字(当前行高的倍数),
+ 如果行高值是数字,则行高会随文字的尺寸改变

技巧 : 
 1.使文字垂直居中 : 将行高属性值 等于 盒子高度属性值
+ 只能使单行文字垂直居中
