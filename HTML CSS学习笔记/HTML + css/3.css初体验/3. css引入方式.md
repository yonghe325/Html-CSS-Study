# 内部样式表 : 学习使用
css代码卸载style标签里面
# 外部样式表 : 开发使用
css代码书写在单独的css文件中(.css)
在HTML中使用link便签引入
```
<link rel = "stylesheet" href="<路径>">
```
+ rel ="<引入种类>"
## css文件书写规范
==在html中 : style标签里面怎么写,在css文件中就怎么写==
# 行内样式 : 配合js使用
css代码写在标签的style属性的 值 里
```
<div style="color: red; font-size: larger;">这是行内引入</div>
```
