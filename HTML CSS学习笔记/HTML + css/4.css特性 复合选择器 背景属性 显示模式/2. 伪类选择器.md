伪类表示元素状态,选中元素的某个状态设置样式
写法 : 选择器 : 元素状态 {CSS属性}
```
</style>
		.box {
            color: blue;
        }
        a {
            color: black;
            text-decoration: none;
        }
        a:hover {
            color: blue;
            text-decoration: underline;    
        }
</style>
</head>

```
# 超链接状态
为超链接书写状态,必须按照顺序写
![[超链接状态.png]]
 + :link : 访问前
 + :visited : 访问后
 + :hover : 鼠标悬停
 + :active : 点击时(激活)
