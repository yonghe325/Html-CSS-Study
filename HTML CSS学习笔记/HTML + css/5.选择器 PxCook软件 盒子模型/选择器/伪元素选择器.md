作用 : 创建虚拟元素(伪元素),用来摆放装饰性内容
E::before : 在E元素里面最前面添加一个伪元素 
E::after : 在E元素里面最后面添加一个伪元素
注意 : 
+ 必须设置content :" "属性,用来设置伪元素的内容,如果没有元素,则引号留空即可
+ 伪元素是行内显示模式
+ 权重和标签选择器相同


```
        div {
            width: 300px;
            height: 300px;
            background-color:pink;
            /* text-align: center;
            line-height: 300px; */
        }
        div::before{
            content: "老鼠";

            display: inline-block;
            width: 100px;
            height: 100px;
            color: red;
            background-color: blue;
        }
        div::after{
            content: "大米";

            display: inline-block;
            width: 100px;
            height: 100px;
            color: red;
            background-color: blue;
        }
    </style>
    <div>爱</div>

  ```
