将常用图标用字体图标代替
字体图标 : ==展示的是图标,本质是***字体***==
作用 : 在网页中加载==简单的,颜色单一==的小图标
优点 : 
1. 灵活
2. 轻量级
3. 兼容性
4. 使用方便
# 使用
## 下载字体
iconfont 图标库 : https://www.iconfont.cn
![[图标库.png]]
登录 -> 素材库 -> 官方图标库 -> 进入图标库 -> 选图标,加入购物车------------------------------> 
购物车,添加至项目,确定 ->下载至本地
## 使用字体
1. link引入字体样式表(iconfont.css)
![[引入字体样式表.png]]
2. 标签使用字体图标类名
	1. iconfont : 字体图标基本样式(字体名,字体大小等)
	2. icon-xxx : 图标对应的类名
![[标签使用字体图标类名.png]]
