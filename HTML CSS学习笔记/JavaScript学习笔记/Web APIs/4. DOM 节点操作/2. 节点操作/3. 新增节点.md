先创建节点
在插入节点
# 创建节点
```js
对象.creatElement('标签名')
```
# 插入节点
```js
//插入到父元素的 最后一个子元素 的 后面
父元素.appendChild(要插入的元素)

//插入到父元素 某一个子元素 的 前面
父元素.insertBefore( 要插入的元素 , 目标元素)
	//要插入的元素 将被插入到 目标元素 的前面
```

# 克隆节点
```js
新元素 = 元素.cloneNode(布尔值)
	//当布尔值为 true  则克隆时会 包含 所有后代节点一起克隆
	//当布尔值为 false 则克隆时会 只会包含子节点 不会包含后代节点 
	//默认为fales
	
```
