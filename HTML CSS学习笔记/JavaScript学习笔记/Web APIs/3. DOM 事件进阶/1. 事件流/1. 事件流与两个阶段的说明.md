+ ==事件流指的是事件完整执行过程中的流动路径==
	1. 捕获阶段
		Document -> Element html ->Element body -> Element div
		事件由大向小流动
		```js
		addEventListener('事件',行为,true/false)
		//当addEventListener的第三个参数为true是
		//则将时间列切换为捕获
		//时间将从由大向小流动
		//为每一个对象执行一次对应的事件行为
			//如果有对象没有该行为, 则跳过
		
		//当addEventListener没有第三个参数时
		//默认为false, 冒泡
		```
	1. 冒泡阶段
		Document <- Element html <- Element body <- Element div
		事件由小向大流动
		当一个事件被触发后
		同样的事件会在该元素的的所有祖先元素中依次被触发
		称为事件冒泡
