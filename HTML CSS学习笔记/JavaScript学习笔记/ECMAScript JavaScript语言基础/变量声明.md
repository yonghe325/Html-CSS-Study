变量声明有三个关键字
+ var let const
	1. var 已被淘汰, 在任何场景下都不要使用
	2. const 和let
		1. const 优先:
			+ const语义化根号
			+ 很多变量在声明时,就已经确定他不会被改变
	3. 对于大多数变量
		1. 在声明时先用const声明
		2. 当需要改变时, 在替换为let
+ 特殊情况
	+ 基本数据类型 与 复杂数据结构类型(数组)
		1. 对于基本数据类型, 当一个变量的==值==不会被修改, 就可以使用const声明
		2. 对于数组等 复杂数据结构类型 和 引用数据类型 来说
			==数组的数据被存储在堆中==
			==而数组名作为一个指针, 存储在栈中==
			对于数组的增删,修改的是位于==堆中的数据==
			没有改变==位于 栈 中的数组名==指向的==堆中数据的位置(地址)==
			<h4>即没有修改数组名指向的地址</h4>
			所以可以使用const声明
