## 影响浮动元素的,实际位置 ,显示位置
负margin会改变浮动元素的显示位置，即使元素写在后面，也可以用负margin值
让它显示在最前面。
```
 <style>
        .box1,.box2,.box3{
            width:100px;
            height:100px;
            float: left;
        }
        .box1{
            background-color: red;
        }
        .box2{
            background-color: yellow;
        }
        .box3{
            background-color:blue;
        }
 </style>
<body>
    <div class="box1">盒子1</div>
    <div class="box2">盒子2</div>
    <div class="box3">盒子3</div>
</body>
```

盒子3完全覆盖盒子1
