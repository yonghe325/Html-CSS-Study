+ 事件对象是什么
	1. 事件对象也是个对象，这个对象里有事件触发时的相关信息
	2. 例如：鼠标点击事件中，事件对象就存了鼠标点在哪个位置等信息
+ 使用场景
	1. 可以判断用户按下哪个键，比如按下回车键可以发布新闻
	2. 可以判断鼠标点击了哪个元素，从而做相应的操作
# 语法
+ 在==事件绑定的回调函数的第一个参数==就是事件对象
```js
      const but = document.querySelector("ul");
      but.addEventListener("click", function (event) {
        console.log(event);
      });
```
+ 一般命名为event, ev, e
# 事件对象常用属性
+ type : 获取当前事件类型
+ clientX/clientY : 获取光标相对于==页面左上角==的位置
+ offsetX/offsetY : 获取光标相对于当前==DOM元素左上角==的位置
+ key = 用户按下键盘的值
