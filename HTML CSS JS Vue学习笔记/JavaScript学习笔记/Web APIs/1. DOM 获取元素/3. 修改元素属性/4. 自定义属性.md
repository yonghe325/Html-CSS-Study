属性分为 标准属性 和 自定义属性
# 标准属性
标签天生自带的属性
比如 class id title等
可以直接使用 `.`  对其进行修改
# 自定义属性
在html5中, 推出了专门的data-自定义属性
在标签上一律以 `data-` 开头
在DOM对象上一律以dataset对象的方式获取
```js
  <div class="box" data-id="10">这是盒子</div>
  
  <script>
    const box = document.querySelector('.box')
    console.log(box.dataset.id);
  </script>
```
## dataset
对一个对象单独使用dataset
```js
		obj.dataset
```
会返回一个子对象==DOMStringMap==
该 子对象的键和值 分别对 母对象 的`data-`自定义属性
共有几个自定义属性, 就有几个键值对
