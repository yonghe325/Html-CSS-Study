# typeof + 变量
可以返回当前变量的数据类型
# 基本数据类型
## number 数字类型
整数,浮点数,统称为数据类型
同时JS作为==弱数据类型==的语言 
不用使用关键字来声明变量
### NaN(Not a number)变量
NaN变量数字类型中的一个特殊类型
==代表一个计算错误==
它是一个不正确的或者一个未定义的数学操作所得到的结果
例如 : 除以零, 一个string与一个数字类型运算
NaN是粘性的
==即任何对NaN的操作都会返回NaN==
==NaN不等于NaN==
## 字符串型
使用单引号(' '), 双引号(" "), 反引号(\` \`)包裹的字符
称为字符串
引号可以相互嵌套(外单内双, 外双内单).
字符串可以进行一系列操作: 
+ 拼接 : 使用 '+' 运算符拼接多个字符串
### 模板字符串(格式打印)
```js
let age = prompt('请输入年龄 : ')
document.write(`你的年龄是&{age}`)
```
使用==反引号==声明声明模板字符串
使用 ${[变量名]} 以字符串的形式 打印对应的变量
## undefined 未定义数据类型
当声明一个变量, 但没有赋值
则默认是undefined类型
## null 空类型
将来有个量里面存放的是一个对象，但是对象还没创建好，可以先给个null
防止将其默认为undefined类型
# 数据类型转换
1. 转换为数字类型
+ 完全转换
```js
let num = Number(prompt('输入 : '))
//或
let num = +prompt('输入 : ')
	//隐式类型转换
```
+ 只保留整数部分
```js
console.log(parseInt('123456.987654321')) 
```
+ 可以保留小数
```js
console.log(parseFloat('123456.987654321')) 
```
2. 截断
	当转换为数字类型时, 只会截取到第一个非数字类型的字符前
```js
    console.log(parseInt('123456.987654321'))
//123456.987654321123456.987654321
    console.log(parseInt('123456abc.987654321'))
//123456
    console.log(parseFloat('123456.abc987654321'))
//123456
    console.log(parseFloat('123456.98a7654321'))
//123456.98
    console.log(parseFloat('123456.987654a321'))
//123456.987654
    console.log(parseFloat('123456.9876a54321'))
//123456.9876
```
