就是模板
类似于c++中的类
+ 不同的是 : 
	1. 匿名类可以被赋值给变量
+ 相同的是 : 
	1. 同样拥有构造函数
	2. 同样有 公共和私有 元素/方法
	3. 都有继承,多态
# 使用
```js
	class Rectangle {
		#height = 0; 
		#width;
//私有元素
	//使用 #开头 进行标识
	  constructor(height, width) {
	    this.#height = height;
	    this.#width = width;
	  }
//构造函数
	//使用this回调自身添加属性	
	// getter 方法
	}
```
