
| Symbol 属性                       | 用途                  | 调用位置                                  | 示例                              |
| ------------------------------- | ------------------- | ------------------------------------- | ------------------------------- |
| **`Symbol.iterator`**           | 定义默认迭代器             | `for...of`、`...`、`yield*`             | `[...obj]`                      |
| **`Symbol.asyncIterator`**      | 定义异步迭代器             | `for await...of`                      | `for await (const x of obj)`    |
| **`Symbol.hasInstance`**        | 自定义 `instanceof`    | `obj instanceof Constructor`          | `[] instanceof MyArray`         |
| **`Symbol.isConcatSpreadable`** | 控制 `concat()` 是否展开  | `arr.concat(obj)`                     | 控制是否打平                          |
| **`Symbol.species`**            | 控制衍生对象的构造函数         | `map()`、`filter()` 等                  | `arr.map(x => x*2).constructor` |
| **`Symbol.match`**              | 自定义 `match()` 行为    | `str.match(obj)`                      | 自定义匹配逻辑                         |
| **`Symbol.replace`**            | 自定义 `replace()` 行为  | `str.replace(obj, replacement)`       | 自定义替换逻辑                         |
| **`Symbol.search`**             | 自定义 `search()` 行为   | `str.search(obj)`                     | 自定义搜索逻辑                         |
| **`Symbol.split`**              | 自定义 `split()` 行为    | `str.split(obj)`                      | 自定义分割逻辑                         |
| **`Symbol.toPrimitive`**        | 自定义类型转换             | 强制类型转换                                | `String(obj)`、`Number(obj)`     |
| **`Symbol.toStringTag`**        | 自定义 `toString()` 标签 | `Object.prototype.toString.call(obj)` | `[object MyClass]`              |
| **`Symbol.unscopables`**        | 控制 `with` 语句行为      | `with (obj) { ... }`                  | 已废弃                             |
|                                 |                     |                                       |                                 |
