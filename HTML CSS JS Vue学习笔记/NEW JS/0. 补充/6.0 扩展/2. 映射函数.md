即对所有元素进行的统一操作
定义了"如何转换每一个元素"的转换方式
# 映射函数
```js
const mappingFunction = (x) => {
    return x ** 2;  // 将每个元素平方
};
// 等价写法：
const mappingFunction2 = function(x) {
    return Math.pow(x, 2);
};
```
# 映射函数参数
```js
const mappingFunction = (x) => {
    return x ** 2;
};
```
映射函数参数即为`x`, **代表被遍历到的数组中的当前元素**
映射函数可以有一或三个参数
分别是当前元素值, 当前索引, 原数组
```js
const arr = [10, 20, 30];
const result = Array.from(arr, 
    (currentValue, index, array) => {
        // currentValue: 当前元素值
        // index: 当前索引
        // array: 原数组
        return `索引${index}的值是${currentValue}`;
    }
);
console.log(result);
// ['索引0的值是10', '索引1的值是20', '索引2的值是30']
```
参数名不重要, 重要的是位置
