## `flat()`方法
+ 传参 : 
	传入打平的层级, 默认不传参为1
	传入`Infinity`时, 打平所有层级
```js
const nested = [1, [2, [3, [4]]]];
console.log(nested.flat());     // [1, 2, [3, [4]]]  (默认打平一层)
console.log(nested.flat(2));    // [1, 2, 3, [4]]    (打平两层)
console.log(nested.flat(Infinity)); // [1, 2, 3, 4]  (完全打平)
```
## `flatMap()`方法
这是 `map()` 和 `flat()` 的组合
它先对每个元素应用一个映射函数，然后将结果打平一层
即修改后在打平
```js
const arr = [1, 2, 3];
// 将每个元素变成 [x, x*10]，然后打平
const result = arr.flatMap(x => [x, x * 10]);
console.log(result); // [1, 10, 2, 20, 3, 30]
```
