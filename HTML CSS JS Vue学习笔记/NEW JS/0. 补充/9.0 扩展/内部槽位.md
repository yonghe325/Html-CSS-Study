内部槽位:
**是 JavaScript 引擎内部用于存储对象状态的特殊属性
不能通过 JavaScript 代码直接访问，只能用规范定义的语言描述**
如 `[[Prototype]]`、`[[Extensible]]`等
常见的内部槽位有

|                     |             |                                         |
| ------------------- | ----------- | --------------------------------------- |
| `[[Prototype]]`     | 所有对象        | 对象的原型（可通过 `Object.getPrototypeOf()` 访问） |
| `[[Extensible]]`    | 所有对象        | 对象是否可扩展（可通过 `Object.isExtensible()` 检查） |
| `[[Get]]`           | 所有对象        | 获取属性的内部方法                               |
| `[[Set]]`           | 所有对象        | 设置属性的内部方法                               |
| `[[Call]]`          | 函数对象        | 函数是否可被调用                                |
| `[[Construct]]`     | 构造函数        | 函数是否可被 `new` 调用                         |
| `[[NumberData]]`    | `Number` 对象 | 存储数值的内部槽位                               |
| `[[StringData]]`    | `String` 对象 | 存储字符串的内部槽位                              |
| `[[DateValue]]`     | `Date` 对象   | 存储时间戳的内部槽位                              |
| `[[RegExpMatcher]]` | `RegExp` 对象 | 存储正则表达式匹配器的内部槽位                         |
| `[[MapData]]`       | `Map` 对象    | 存储键值对数据的内部槽位                            |
| `[[SetData]]`       | `Set` 对象    | 存储集合数据的内部槽位                             |
内部槽位在对象创建时直接由引擎分配
且只能由引擎使用
所以对象不同, 内部槽位也不同
## `[[Extensible]]`
### `isExtensible()`方法
查询对象是否可扩展
对应的捕获器转到[[10. isExtensible()]]
### `preventExtensions()`方法
禁止对象扩展
对象的捕获器转到[[11. preventExtensions()]]
### `Object.seal()`方法
密封对象, 将`[[Extensible]]`设为`false`
### `Object.freeze`方法
冻结对象, 将`[[Extensible]]`设为`false`
# 属性级别的内部槽位
|                |                    |                     |
| -------------- | ------------------ | ------------------- |
| `configurable` | `[[Configurable]]` | 属性是否可配置（可删除、可修改描述符） |
| `enumerable`   | `[[Enumerable]]`   | 属性是否可枚举             |
| `writable`     | `[[Writable]]`     | 属性值是否可写（仅数据属性）      |
| `value`        | `[[Value]]`        | 属性的值（仅数据属性）         |
| `get`          | `[[Get]]`          | getter 函数（仅访问器属性）   |
| `set`          | `[[Set]]`          | setter 函数（仅访问器属性）   |
