"中断"是计算机系统中的一个底层术语，指 CPU 暂停当前正在执行的任务
转而去处理一个更紧急或预先安排好的任务，处理完后再回到原来的任务
# 异常
"异常" 通常是指程序运行出错
 而不是人为导致的错误
 通过`throw`关键字抛出异常, 会终止抛出异常后的所有行为
 包括同步操作 与 异步操作
# 同步错误 与 异步错误
错误(Error),也是一种类型, 其实例称为错误对象 
一个错误可以通过`Error()`构造函数实例化
传入构造参数一个参数代表该错误的原因
使用`throw`关键字抛出错误, 同样会中断行为

# 捕获错误
异常 与 错误 都可以被 `try/catch`捕获, 用于发生错误后, 运行相对应的程序
在异步操作抛出的异常/错误, 不会被同步操作中的`try/catch`捕获
