内嵌窗格指inline frame
允许当前网页中嵌入另一个HTML文档
# 基本概念
```html
<!-- 最简单的 iframe -->
<iframe src="https://example.com"></iframe>

<!-- 带属性的 iframe -->
<iframe 
  src="https://example.com" 
  width="800" 
  height="600" 
  frameborder="0">
</iframe>
```
+ 嵌入的是一个完整的独立文档
+ 有自己的window, document, 历史记录
+ 样式和脚本不会干扰父页面
# 常用属性
+ `src`: 嵌入页面的URL
+ `srcdoc` : 直接嵌入的DOM树字符串
+ `width/height`尺寸
+ `name` : 名称, 用于链接跳转
+ `sandbox` : 安全权限
	`sandbox="allow-scripts"`
+ `allow` : 权限控制
	`allow="camera; microphone"`
+ `loading` : 懒加载
	`loading="lazy"`
