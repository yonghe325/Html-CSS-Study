`NodeFilter` 是一个用于**过滤节点**的接口，配合 `TreeWalker` 或 `NodeIterator` 使用
可以精确控制需要遍历的节点类型

`NodeFilter` 允许定义**哪些节点应该被包含或排除**，实现**自定义节点过滤逻辑**
```js
// 基本结构
const filter = {
  acceptNode(node) {
    // 返回 NodeFilter.FILTER_ACCEPT 或 NodeFilter.FILTER_REJECT 或 NodeFilter.FILTER_SKIP
  }
};
```
因为 NodeFilter 是一个抽象类型，所以不可能创建它的实例
所以只需要创建一个包含`acceptNode()`的方法的对象,
然后传给对应的`createNodeIterator()`方法或`createTreeWalker()`方法

---
`NodeFilter`有且仅有三个返回值:
+ `NodeFilter.FILTER_ACCEPT` : 掩码为1
	代表接受该节点
+ `NodeFilter.FILTER_REJECT` : 掩码为2
	拒绝该节点, 同时跳过其子树
	仅能在`TreeWalker`中使用
+ `NodeFilter.FILTER_SKIP` : 掩码为3
	直接跳过该节点
