css样式有四种类型
# 原始值(1)
这是最简单、最基础的类型。就是一个值，不能再拆分了。
- **CSS 例子**：`color: red;`、`border: none;`、`display: block;`、`width: 100px;`
# 列表(2)
有些 CSS 属性需要提供一组值，这些值按顺序排成列表。
- **CSS 例子**：`font-family: "微软雅黑", Arial, sans-serif;` (列表)
	`margin: 10px 20px 10px 20px;` (列表)
	`background: url(a.png) no-repeat center;` (也是列表)
# 自定义值(3)
个类型主要用来表示 **CSS 自定义属性（CSS变量）** 或某些未被标准化的值。
- **CSS 例子**：`var(--main-color)` 或者像 `width: 一些目前浏览器还不支持的未来语法`
# 继承的值(0)
这个最特别。它不是值本身，而是表示 **“这个属性的值将从它的父元素那里继承过来”**。
- **CSS 例子**：你没有给一个 `<p>` 标签写 `color` 属性，浏览器就会说它的 `color` 是 `inherit`。
