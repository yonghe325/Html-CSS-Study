ECMAScrip 里的函数传参, ==都是按值传递==
区别是传的 原始值 还是 引用值
+ 原始值传值, 只会传递该值的复制
+ 引用型传址, 与引用值变量==复制==机制一致
	这就代表引用值变量 的形参与实参是两个变量, 只不过指向同一个对象(地址)
	在函数内,==在未改变形参地址的前提下==, 通过修改形参可以影响实参
	```js
	function setName(obj) {
		obj.name = "Nicholas";
		obj = new Object();
		obj.name = "Greg";
	}
	
	let person = new Object();
	setName(person);
	console.log(person.name); // "Nicholas"
	```
