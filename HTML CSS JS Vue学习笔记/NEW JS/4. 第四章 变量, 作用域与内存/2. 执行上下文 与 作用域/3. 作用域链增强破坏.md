# eval()
[eval()](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/eval)函数的功能是动态执行语句
```js
eval(<被传入的js代码>)
console.log(eval("2 + 2"));
// Expected output: 4
console.log(eval(new String("2 + 2")));
// Expected output: 2 + 2
console.log(eval("2 + 2") === eval("4"));
// Expected output: true
console.log(eval("2 + 2") === eval(new String("2 + 2")));
// Expected output: false
```
>永远不要使用`eval()`

它会改变函数的的作用域链, 还会使程序更危险
完全可以使用`Function()`函数来代替
# with语句
[[4. with 语句]]
`with`语句会改变对应变量/ 函数的作用域链
会将对应的作用域对应变量的作用域链的最前端
会让代码难以理解
# try/catch语句
`try/catch`语句中的`catch`块会创建新的作用域
会把`try`抛出的错误对象接收并重命名后
将接收后的错误对象 放到 新创建的的作用域里
并将该作用域放到当前作用域的最前端
当使用一个变量时, 就会先在`catch`作用域内寻找
找不到再向上级作用域
