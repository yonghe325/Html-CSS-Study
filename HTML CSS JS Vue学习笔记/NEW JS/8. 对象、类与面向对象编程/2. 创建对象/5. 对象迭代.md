对象默认是不可迭代的, 除非手动为其添加迭代器
ECMA2017后, 
可以使用`Object.values()`方法  与 `Object.entries()`方法
将对象转化为数组, 使其可迭代
+ `Object.values()`方法
	返回对象值的数组
+ `Object.entries()`方法
	返回对象属性键/值对的数组

```js
const o = {
	foo: 'bar',
	baz: 1,
	qux: {}
};
console.log(Object.values(o));
// ["bar", 1, {}]
console.log(Object.entries((o)));
// [["foo", "bar"], ["baz", 1], ["qux", {}]]
```
注意，非字符串属性会被转换为字符串输出
这两个方法执行对象的浅复制
```js
const o = {
	qux: {}
};
console.log(Object.values(o)[0] === o.qux);
// true
console.log(Object.entries(o)[0][1] === o.qux);
// true
```
符号属性会被忽略
```js
const sym = Symbol();
const o = {
	[sym]: 'foo'
};
console.log(Object.values(o));
// []
console.log(Object.entries((o)));
// []
```
