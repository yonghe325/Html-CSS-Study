ECMAScript 6 新引入的 class 关键字具有正式定义类的能力
	被用作创建对象的蓝图/模板
类（class）是ECMAScript 中新的基础性语法糖结构
它的本质是一个函数, 整合了 构造函数 + 原型对象 的功能
例如
```js
// 类
class Person {
    constructor(name) {
        this.name = name;
    }
    sayHi() {
        console.log(`Hi, I'm ${this.name}`);
    }
}

// 等价的构造函数写法
function Person(name) {
    this.name = name;
}
Person.prototype.sayHi = function() {
    console.log(`Hi, I'm ${this.name}`);
};

// 两者本质上是一样的！
console.log(typeof Person);  // "function"（两种写法都返回 "function"）
```
---
# 定义类
类的定义与函数类似, 有 类声明, 类表达式两种声明方式
都有几点与函数不同
+ ==类无法被提升==
```js
console.log(FunctionExpression); // undefined
var FunctionExpression = function() {};
	//这里是因为var 只提升声明 不提升值
console.log(FunctionExpression); // function() {}

console.log(FunctionDeclaration); // FunctionDeclaration() {}
	//被提升
function FunctionDeclaration() {}
console.log(FunctionDeclaration); // FunctionDeclaration() {}
console.log(ClassExpression); // undefined
	//var将class的声明提升, 没提升值
var ClassExpression = class {};
console.log(ClassExpression); // class {}

console.log(ClassDeclaration); // ReferenceError: ClassDeclaration is not defined
class ClassDeclaration {}
console.log(ClassDeclaration); // class ClassDeclaration {}
```
 + 类会被块作用域限制, 而函数不会
```js
function FunctionDeclaration() {}
	class ClassDeclaration {}
}
console.log(FunctionDeclaration); // FunctionDeclaration() {}
console.log(ClassDeclaration); // ReferenceError: ClassDeclaration is not defined
```

#  类的构成
>类可以包含构造函数方法、实例方法、获取函数、设置函数和静态类方法
但这些都不是必需的
空的类定义照样有效
```js
// 空类定义，有效
class Foo {}
// 有构造函数的类，有效
class Bar {
	constructor() {}
}
// 有获取函数的类，有效
class Baz {
	get myBaz() {}
}
// 有静态方法的类，有效
class Qux {
	static myQux() {}
}
```
类的表达式名称是可选的
在把类表达式赋值给变量后
可以通过 `name` 属性取得类表达式的名称字符串(即标识符)
但不能在==类表达式作用域外部访问这个标识符==
仅能在类的内部使用该标识符进行回调
```js
let Person = class PersonName {
	identify() {
		console.log(Person.name, PersonName.name);
	}
}
let p = new Person();
p.identify(); // PersonName PersonName
console.log(Person.name); // PersonName
	//name返回标识符
console.log(PersonName); // ReferenceError: PersonName is not defined
	//该标识符在类的外部不能使用
```
