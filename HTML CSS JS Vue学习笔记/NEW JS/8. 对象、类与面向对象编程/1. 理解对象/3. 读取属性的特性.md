# `Object.getOwnPropertyDescriptor()`方法
通过使用`Object.getOwnPropertyDescriptor()`方法
可以取得指定属性的属性描述符(特性)
```js
Object.getOwnPropertyDescriptor(<目标对象>,<目标对象的一个属性>);
```
+ 返回值 : 
	返回一个对象, 属性为`目标属性`的所有特性
	`[[configurable]]`为`true/false`
	`[[Enumerable]]`为`true/false`
	`[[Writable]]`为`true/false`
		如果`目标属性`为访问器 属性, 则没有该属性, 返回`Undefined`
	`[[value]]`为`目标属性`的值
	 值的类型与目标属性的值的类型相同
	`[[Get]]`为指向`目标属性`获取函数的一个指针
	`[[Set]]`为指向`目标属性`设置函数的一个指针
# `Object.getOwnPropertyDescriptors()`方法
作用于`Object.getOwnPropertyDescriptor()`方法类似
```js
Object.getOwnPropertyDescriptors(<目标对象>)
```
+ 返回值 : 
	返回一个对象:
	这个对象的属性 为`目标对象`的所有属性名
	这个对象的属性 的属性值为`目标对象`对应对应属性的所有特性描述
```js
let book = {};
Object.defineProperties(book, {
	year_: {
		value: 2017
	},
	edition: {
		value: 1
	},
	year: {
		get: function() {
			return this.year_;
		},
		set: function(newValue){
			if (newValue > 2017) {
				this.year_ = newValue;
				this.edition += newValue - 2017;
			}
		}
	}
});
console.log(Object.getOwnPropertyDescriptors(book));
// {
	// edition: {
		// configurable: false,
		// enumerable: false,
		// value: 1,
		// writable: false
	// },
	// year: {
		// configurable: false,
		// enumerable: false,
		// get: f(),
		// set: f(newValue),
	// },
	// year_: {
		// configurable: false,
		// enumerable: false,
		// value: 2017,
		// writable: false
	// }
// }
```
