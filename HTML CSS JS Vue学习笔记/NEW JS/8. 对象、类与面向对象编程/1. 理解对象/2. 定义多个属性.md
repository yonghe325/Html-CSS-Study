通过`Object.defineProperties()`方法(区别与`Object.definePropertie()`方法)
可以为对象一次性==添加/修改==多个==数值/ 访问器属性==
```js
Object.defineProperties(<目标对象>,{
	<要添加/修改 的 属性1:{
		[特性1 : 值1],......
	}>
	<要添加/修改 的 属性2:{
		[特性1 : 值1],......
	}>
	<要添加/修改 的 属性3:{
		[特性1 : 值1],......
	}>
	<要添加/修改 的 属性3:{
		[特性1 : 值1],......
	}>
	....
})
```
