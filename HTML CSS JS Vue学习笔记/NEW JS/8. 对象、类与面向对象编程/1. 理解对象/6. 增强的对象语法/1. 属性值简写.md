>ECMA6为定义与操作对象添加了许多极其有用的语法糖特性
被称为增强的对象语法
# 属性值简写
当属性名与属性值的变量名一致时
可以使用该与语法糖
```js
let name = 'Matt';
let person = {
	name: name//这里属性名为name, 属性值是一个变量, 变量名为name
};
console.log(person); // { name: 'Matt' }
/** 等价于 **/

let name = 'Matt';
let person = {
	name//只写属性名
};
console.log(person); // { name: 'Matt' }
```
代码压缩程序会在不同作用域间保留属性名，以防止找不到引用。以下面的代码为例：
```js
function makePerson(name) {
	return {
		name
	};
}
let person = makePerson('Matt');
console.log(person.name); // Matt
```
多次调用`makePerson()`函数, 只会以字符串的形式保留一次属性名
即多个在`makePerson()`创建的对象 ==共用同一个保留属性名==
在执行[[代码压缩程序Code Minifier|代码压缩程序]]时,函数参数会被缩短
由 : 
```js
function makePerson(name) {
	return {
		name
	};
}
let person = makePerson('Matt');
console.log(person.name); // Matt
```
变为:
```js
function makePerson(a) {
	return {
		name : a
	};
}
let person = makePerson('Matt');
console.log(person.name); // Matt
```
