因为弱键的存在, `WeakMap`不会阻碍垃圾回收机制
所以`WeakMap`非常适合用来存储Dom节点
## 对于`Map来说`
```js
const m = new Map();
const loginButton = document.querySelector('#login');
// 给这个节点关联一些元数据
m.set(loginButton, {disabled: true});
```
当`'#login'`对应的结点被删除时, 对应的DOM结点仍然在内存中
不会被回收
## 对于`WeakMap`来说
```js
const wm = new WeakMap();
const loginButton = document.querySelector('#login');
// 给这个节点关联一些元数据
wm.set(loginButton, {disabled: true});
```
当节点从 DOM 树中被删除后，垃圾回收程序就可以立即释放其内存
（假设没有其他地方引用这个对象）
