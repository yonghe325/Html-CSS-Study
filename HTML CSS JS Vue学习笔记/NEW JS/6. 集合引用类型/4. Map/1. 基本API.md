即哈希映射
# 初始化
```js
// 使用嵌套数组初始化映射
const m1 = new Map([
	["key1", "val1"],
	["key2", "val2"],
	["key3", "val3"]
]);
alert(m1.size); // 3

// 使用自定义迭代器初始化映射
const m2 = new Map({
	[Symbol.iterator]: function*() {
		yield ["key1", "val1"];
		yield ["key2", "val2"];
		yield ["key3", "val3"];
	}
});
alert(m2.size); // 3
```
# 增删改查
## `set(<键> , <值>)`
添加映射, 多个set可以使用`.`连接
`<键>`可以是任何JavaScript数据类型
只要不是相同的实例 , 就不会冲突
即 ==SameValueZero==
```js
const m = new Map();
const functionKey = function() {};
const symbolKey = Symbol();
const objectKey = new Object();

m.set(functionKey, "functionValue");
m.set(symbolKey, "symbolValue");
m.set(objectKey, "objectValue");
alert(m.get(functionKey)); // functionValue
alert(m.get(symbolKey)); // symbolValue
alert(m.get(objectKey)); // objectValue
// SameValueZero 比较意味着独立实例不冲突
alert(m.get(function() {})); // undefined
	
const m = new Map();
const a = 0/"", // NaN
b = 0/"", // NaN
pz = +0,
nz = -0;

//SameValueZero可能会导致奇怪的冲突
alert(a === b); // false
alert(pz === nz); // true

m.set(a, "foo");
m.set(pz, "bar");
alert(m.get(b)); // foo
alert(m.get(nz)); // bar
```
## `delete(<键>)`
删除该`<键>`所对应的映射
## `clear()`
删除该`Map`所有映射
## `has(<键>)`
判断该`<键>`是否存在
## `get(<键>)`
得到该`<键>`对应的值
## `size()`
返回该映射的长度
