Object类型是最常用的一种数据类型
其本身并没有太多方式, 但十分适合用来存储数据
# 声明 与 表达式上下文
## 使用Object构造函数 显式声明
```js
let person = new Object();
//该方式完全等同于let person = {}
person.name = "Nicholas";
person.age = 29;
```
## 对象字面量
```js
let person = {
	name: "Nicholas",
	age: 29
};
```
在上述的例子中, ` { ` 表示对象字面量的开始
因为他出现在==表达式上下文==中
在ECMAScript中,**表达式上下文指的是期待返回值的上下文**
赋值操作符表示后面要期待一个值
因此左大括号表示一个表达式的开始
同样是` { `, 如果出现在语句上下文中, 则表示一个语句块的开始
# 属性调用
## 点语法
在对象后面加上` . `, 再加上属性名, 即可调用该对象的公共属性
```js
console.log(person.name); // "Nicholas"
```
点语法是首选, 除非在调用对象属性时需要使用变量
## 中括号索引法
在对象后面加上`[]`
`[]`中放入对象属性的**字符串形式**
即可调用该属性
```js
console.log(person["name"]); // "Nicholas"
```
**使用中括号的主要优势就是可以通过变量访问属性**
```js
let propertyName = "name";
console.log(person[propertyName]); // "Nicholas"
```
由于属性名中包含可能导致语法错误的字符
```js
person["first name"] = "Nicholas";
//不能使用点语法, 因为"first name"属性名中包含一个空格
```
