遍历迭代数组的所有项
并接收所有元素传入函数的返回值
并将其返回
## `reduce()`方法 与 `reduceRight()`方法
`reduce()`从前向后遍历 , `reduceRight()`从后向前遍历
```js
reduce((<前一个元素>,<当前元素>,<当前元素的下标>,<被操作的数组>) => {<执行操作>})
reduceRight((<前一个元素>,<当前元素>,<当前元素的下标>,<被操作的数组>) => {<执行操作>})
例子1 :
let values = [1, 2, 3, 4, 5];
let sum = values.reduce((prev, cur, index, array) => prev + cur);
alert(sum); // 15

例子2 : 
let values = [1, 2, 3, 4, 5];
let sum = values.reduceRight(function(prev, cur, index, array){
	return prev + cur;
});
alert(sum); // 15
```
