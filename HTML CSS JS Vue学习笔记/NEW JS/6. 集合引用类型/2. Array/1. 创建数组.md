# 构造方式
## 使用Array构造函数
```js
let colors = new Array();
```
构造函数`Array()`可以传入**一个数值**
用以指定数组的初始长度
```js
let colors = new Array(20);
//初始化了一个长度为20的数组
```
或者直接向函数内传入需要存储的数据
```js
let colors = new Array("red", "blue", "green");
//初始化的一个长度为3, 元素分别是"red", "blue", "green"的数组
let colors = new Array(1,2,3);
//创建了一个一个长度为3, 元素分别是1,2,3的数组
```
当传入一个整数数字时, `Array()`函数只会将其解析为长度, 不会初始化包含该元素的数组
```js
// 创建一个包含 3 个元素的数组
let names = new Array("Greg"); 
// 创建一个只包含一个元素，即字符串"Greg"的数组
```
当使用`Array()`函数创建数组时, 可以省略`new`操作符
```js
let colors = Array(3); 
let names = Array("Greg"); 
```
## 数组字面量
直接使用中括号创建一个数组
```js
let colors = ["red", "blue", "green"]; 
// 创建一个包含 3 个元素的数组
let names = []; 
// 创建一个空数组
let values = [1,2,]; 
// 创建一个包含 2 个元素的数组
```

## `Array.form()`函数
`Array.form()`函数的作用是将所有类数组结构
即[[1. 可迭代对象 结构|可迭代的结构]], 或者==有length属性==和可索引的结构
例如string类型数据,map和set等结构
转换为数组结构
```js
//将字符串转换为数组
console.log(Array.from("Matt")); 
// ["M", "a", "t", "t"]
------------------------------------------
//将集合和映射(哈希表)转换为数组
const m = new Map().set(1, 2).set(3, 4);
console.log(Array.from(m)); // [[1, 2], [3, 4]]
const s = new Set().add(1).add(2).add(3).add(4);
console.log(Array.from(s)); // [1, 2, 3, 4]
------------------------------------------
//适用于任何可迭代的结构, 甚至是自定义迭代对象
const iter = {
*[Symbol.iterator]() {
	yield 1;
	yield 2;
	yield 3;
	yield 4;
	}
};
//此时iter为可迭代对象
console.log(Array.from(iter)); // [1, 2, 3, 4]
//Symbol.iterator, yield, 自定义迭代在第七章会讲
```
`Array.form()`函数还可以对一个数组进行浅复制
```js
const a1 = [1, 2, 3, 4];
const a2 = Array.from(a1);
//a2是a1的浅复制
console.log(a1); // [1, 2, 3, 4]
alert(a1 === a2); // false
```
`Array.from()`也能转换带有必要属性的自定义对象
```js
const arrayLikeObject = {
	0: 1,
	1: 2,
	2: 3,
	3: 4,
	//必要属性1 : 非负整数索引
		//可以是字符类型:"1"
		//索引可以不连贯:1 3 5 7 88 99
	length: 4
	//必要属性2 : length属性, 值必须是非负整数
};
console.log(Array.from(arrayLikeObject)); 
// [1, 2, 3, 4]
//注意 : 
//1. 转换后的数组的索引与原索引没有任何关系
	//原索引知识from函数将其转换为数组的标识
//2. 当索引个数大于length时
	//from函数会舍弃多于length规定外的元素
```
### `Array.from()`可选增强写法
即在将可迭代转换为数组时, 直接操作(增强)新数组内的元素
不用在使用中间数组进行运算
+ 当`Array.from()`函数传入一个值时
	即普通用法
+ 当`Array.from()`函数传入两个值时
	第一个值依旧是可迭代对象
	第二个值为[[2. 映射函数|映射函数]]参数
		这个函数可以直接增强新数组的值
		如果此时在映射函数内部使用`this`
			在严格模式下, this为`Undefined`
			在非严格模式下, this指向全局对象
```js
const a1 = [1, 2, 3, 4];
const a2 = Array.from(a1, x => x**2);
console.log(a2); // [1, 4, 9, 16]
```
+ 当`Array.from()`函数传入三个值时
	第一个值依旧是可迭代对象
	第二个值为[[2. 映射函数|映射函数]]参数
	第三个为映射函数指定this
```js
const a1 = [1, 2, 3, 4];
const a3 = Array.from(a1, function(x) {return x**this.exponent}, {exponent: 2});
console.log(a3); // [1, 4, 9, 16]
//此时this指向{exponent: 2}
```
## `Array.of()`函数
可以传入一组参数
将传入的参数转换为数组
```js
console.log(Array.of(1, 2, 3, 4)); // [1, 2, 3, 4]
console.log(Array.of(undefined)); // [undefined]
```
