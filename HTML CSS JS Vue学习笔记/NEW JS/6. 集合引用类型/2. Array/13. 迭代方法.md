ECMAScript 为数组定义了 5 个迭代方法
```js
let result  = Method((<当前被查找元素>,<当前元素的下标>,<被查找的数组>) => <判断条件>)
//即
let result  = Method( (item, index, array) => <执行操作> )
```
+ 每个方法接收两个参数: 
	+ 每一项参数都运行的==函数==
	+ 可选的作为函数运行上下文的 ==作用域对象==(影响函数中 this 的值)
## `every()`方法
对数组每一项都运行传入的函数
如果对每一项函数都返回 true，则这个方法返回 true
## `some()`方法
对数组每一项都运行传入的函数
如果有一项函数返回 true，则这个方法返回 true
```js
let numbers = [1, 2, 3, 4, 5, 4, 3, 2, 1];
let everyResult = numbers.every((item, index, array) => item > 2);
alert(everyResult); // false
let someResult = numbers.some((item, index, array) => item > 2);
alert(someResult); // true
```
## `filter()`方法
对数组每一项都运行传入的函数
函数返回 true 的项会组成数组之后返回
```js
let numbers = [1, 2, 3, 4, 5, 4, 3, 2, 1];
let filterResult = numbers.filter((item, index, array) => item > 2);
alert(filterResult); // 3,4,5,4,3
```
## `forEach()`方法
对数组每一项都运行传入的函数
没有返回值
==就是for循环!!!==
```js
let numbers = [1, 2, 3, 4, 5, 4, 3, 2, 1];
numbers.forEach((item, index, array) => {
	// 执行某些操作
});
```
## `map()`方法
对数组每一项都运行传入的函数
返回由每次函数调用的结果构成的数组
```js
let numbers = [1, 2, 3, 4, 5, 4, 3, 2, 1];
let mapResult = numbers.map((item, index, array) => item * 2);
alert(mapResult); // 2,4,6,8,10,8,6,4,2
```
