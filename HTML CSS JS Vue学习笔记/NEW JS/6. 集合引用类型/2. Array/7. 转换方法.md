[[0.1 重写toLocaleString()、toString()和 valueOf()]]
对数组使用`toString()`,`toLocaleString()`与`valueOf()`
都会返回到一个逗号分隔的数组值的字符串
```js
let colors = ["red", "blue", "green"]; // 创建一个包含 3 个字符串的数组
alert(colors.toString()); // red,blue,green
alert(colors.valueOf()); // red,blue,green
alert(colors); // red,blue,green
```
## `join()`方法
传入一个字符串,返回包含所有项的字符串
得到与调用`toString()`相同的结果, 只是把逗号换为指定字符串
```js
let colors = ["red", "green", "blue"];
alert(colors.join(",")); // red,green,blue
alert(colors.join("||")); // red||green||blue
```
+ 不修改字符串
	所指定的分隔符仅仅被这次`join()`方法临时使用
+ `join()`如果不传入参数, 或者传入`Undefined`
	则默认使用`,`作为分隔符
# 区别
## `toString()`,`toLocaleString()`向下委托
对一个`Array`使用`toString()`,`toLocaleString()`
会在内部逐个调用每个元素的`toString()`,`toLocaleString()`
并将其返回值组成为由`,`分割的字符串
所以对单独一个元素重写`toString()`,`toLocaleString()`的返回值
都会体现在整个数组身上
```js
let person1 = {
	toLocaleString() {
	return "Nikolaos";
	},
	toString() {
	return "Nicholas";
	}
};

let person2 = {
	toLocaleString() {
	return "Grigorios";
	},
	toString() {
	return "Greg";
	}
};
let people = [person1, person2];
alert(people); // Nicholas,Greg
alert(people.toString()); // Nicholas,Greg
alert(people.toLocaleString()); // Nikolaos,Grigorios
```

## `valueof()`不会向下委托
不会对数组内的元素逐个使用`valueof()`
而是直接返回数组本身
==即返回一个`Array`对象(object)==
