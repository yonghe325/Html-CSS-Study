# 队列
提供几个方法
使`Array`作为类似于queue(队列)或deque(双端队列)的数据结构
## 方法
### `push()`, 尾插, 返回长度
队尾入队
可以传入多个参数
从前向后一次入队
### `unshift()`头插, 返回长度
头部入队, 返回队列的长度(==有返回值==)
可以传入多个参数
==从后向前依次入队==
```js
let colors = new Array(); // 创建一个数组
let count = colors.unshift("red", "green");
count = colors.unshift("black");
//colors = ["black","red","green"]
```
### `pop()`, 尾删, 返回尾部
取出队尾, 并将其出队
### `shift()`, 头删, 返回头部
取出头部, 并将其出队
