ECMAScript 提供两类搜索数组的方法
按==严格相等搜索==和按==断言函数搜索==
# 严格相等搜索
搜索对象 与 被搜索对象 必须完全相等(\=\=\=)
## `indexOf()`方法 与 `lastindexOf()`方法
```js
indexOf(<查找元素>[,<起始下标>])
lastindexOf(<查找元素>[,<起始下标>])
```
+ 传参 : `<查找元素>` , `<起始下标>`
+ 返回值 : 
	`indexOf()`方法返回从`<起始下标>`开始, ==从前向后==查找第一个符合元素的==下标==
	`lastindexOf`方法返回从`<起始下标>`开始, ==从后向前==查找第一个符合元素的==下标==
	没有符合的元素返回`-1`
## `includes()`方法
判断指定元素是否在指定范围内存在
```js
includes(<查找元素>[,<指定下标>])
```
+ 传参 : `<查找元素>` , `<起始下标>`
+ 返回值 : 
	从`<指定下标>`开始, ==从前向后==查找
	找到返回`true`
	没找到返回`false`

# 断言函数搜索
查找符合要求的元素, 当要求为全等时(\=\=\=)时
作用于[[#严格相等搜索|严格相等搜索]]相同
## `find()`方法 与 `findIndex()`方法
 `find()`方法 与 `findIndex()`方法
 分别返回==第一个== 符合元素  与 符合元素的下标
 ```js
 find( (<当前被查找元素>,<当前元素的下标>,<被查找的数组>) => <判断条件> )
 //(<当前被查找元素>,<当前元素的下标>,<被查找的数组>) => <判断条件>
 //被称为断言函数
 //断言函数用来判断当前被查找的元素是否符合要求
 const people = [
{
	name: "Matt",
	age: 27
	},
	{
	name: "Nicholas",
	age: 29
	}
];
alert(people.find((element, index, array) => element.age < 28));
// {name: "Matt", age: 27}
alert(people.findIndex((element, index, array) => element.age < 28));
// 0
 ```
 + 返回值 : 
	找到返回对应的元素 或 下标
