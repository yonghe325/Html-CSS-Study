以其他语言类似
ECMAScript也是使用`[<索引>]`的方式访问数组元素
不同的是
## 长度动态调整
当使用`[]`修改数组元素时, 可以给出大于数组长度的索引
此时数组自动修改其长度, 将其长度延长至指定为索引
并在最后一个索引位置上==添加==指定的元素
中间扩展的索引将会称为数组空位
此时数组自动修改其长度, 将其长度延长至 **指定索引 + 1**
```js
let colors = ["red", "blue", "green"]; 
// 创建一个包含 3 个字符串的数组
colors[99] = "black";
// 添加一种颜色（位置 99）
alert(colors.length); 
// 100
```
所以ECMAScript的数组可以很方便的使用其length
```js
let colors = ["red", "blue", "green"]; 
// 创建一个包含 3 个字符串的数组
colors[colors.length] = "black"; 
// 添加一种颜色（位置 3）
colors[colors.length] = "brown"; 
// 再添加一种颜色（位置 4）
//类似于push
```
当索引为负整数时,不能访问数组里的元素
当如果有属性名为负整数, 可以访问对应的属性值
```js
const arr = ['a', 'b', 'c'];
arr[-1] = '负索引';  
// 不会成为数组元素！
console.log(arr);     
// ['a', 'b', 'c']
//-1被当作普通属性添加
console.log(arr[-1]); 
// '负索引'（作为属性存在，不是数组元素）
```
## 长度可直接修改
ECMAScript中数组的length
即长度不是只读的, 可以被直接修改
当将长度扩大时, 新增的索引将会成为数组空位
当将长度缩小时, 多余的索引将会消失
