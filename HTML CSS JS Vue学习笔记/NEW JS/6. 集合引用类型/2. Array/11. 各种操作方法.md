# `concat()`方法
```js
let Array1 = {1,2,3}
let Array2 = Array1.concat(数组/元素,......);
```
以原有数组的全部元素为基础, 创建新的数组
+ 传参 :  
	传入 一个或多个 元素或数组
		元素 直接放到新数组的末尾
		数组 将被[[3. 打平|打平]]第一层嵌套,按序放到新数组中
+ 返回值
	返回一个操作后`Array`
```js
let colors = ["red", "green", "blue"];
let colors2 = colors.concat("yellow", ["black", "brown"]);
console.log(colors); // ["red", "green","blue"]
console.log(colors2); // ["red", "green", "blue", "yellow", "black", "brown"]
```
### 取消`concat()`的打平
对传入数组添加特殊的符号`[Symbol.isConcatSpreadable]`
并将其设置为true, `concat()`在解析参数的时候, 就不会将该数组打平
```js
let colors = ["red", "green", "blue"];
let newColors = ["black", "brown"];
let moreNewColors = {
	[Symbol.isConcatSpreadable]: true,
	length: 2,
	0: "pink",
	1: "cyan"
};
newColors[Symbol.isConcatSpreadable] = false;
// 强制不打平数组
let colors2 = colors.concat("yellow", newColors);
// 强制打平类数组对象
let colors3 = colors.concat(moreNewColors);
console.log(colors); // ["red", "green", "blue"]
console.log(colors2); // ["red", "green", "blue", "yellow", ["black", "brown"]]
console.log(colors3); // ["red", "green", "blue", "pink", "cyan"]
```
## `slice()`方法
截取原有数组的部分, 返回新的数组
不修改原数组
```js
let Array1 = {1,2,3,4,5}
let Array2 = Array1.slice(<起始下标>[,<结束下标>])
//从<起始下标>截取指定的<元素个数>
```
+ 传参 : 
	`起始下标`, `结束下标`
		`结束下标`小于等于数组长度, 否则只会截取到数组末尾
## `splice()`方法
对原数组进行修改, 返回被删除的元素数组
```js
let Array1 = {1,2,3,4,5}
let Array2 = Array1.splice(<起始下标>[,<元素个数>[,<元素1>[,元素2...]]])
//可用负数下标
```
+ 传参 : 
	`起始下标`, `元素个数`, `其他元素`
		`元素个数`默认为从起始下标到数组结尾的所有元素
			`元素个数`大于最大个数时, 只操作安全的下标
		`其他元素`将被插入到`起始下标`之后
			插入方式为浅拷贝, 不会被打平
+ 返回值 : 
	返回以被删除的元素组成的数组
使用`splice()`可以间接的对数组进行
+ 删除 : 传入两个参数
	指定要删除的第一个元素的位置 和 要删除的元素数量
	```js
		let colors = ["red", "green", "blue"];
		let removed = colors.splice(0,1); // 删除第一项
		alert(colors); // green,blue
		alert(removed); // red，只有一个元素的数组
	```
+ 插入 : 传入三个参数
	开始位置、0（要删除的元素数量）和要插入的元素
	```js
		alert(colors); // green,blue	
		removed = colors.splice(1, 0, "yellow", "orange"); // 在位置 1 插入两个元素
		alert(colors); // green,yellow,orange,blue
		alert(removed); // 空数组
	```
	+ 替换 : 传入三个参数
	开始位置、要删除元素的数量和要插入的任意多个元素
	先删除, 后添加
	```js
		alert(colors); // green,yellow,orange,blue
		removed = colors.splice(1, 1, "red", "purple"); // 插入两个值，删除一个元素
		alert(colors); // green,red,purple,orange,blue
		alert(removed); // yellow，只有一个元素的数组
	```
