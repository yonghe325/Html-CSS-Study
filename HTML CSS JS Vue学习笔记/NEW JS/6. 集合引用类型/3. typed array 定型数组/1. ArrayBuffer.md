一种表示 **原始二进制数据缓冲区** 的对象
即用来存储二进制数据的对象
传入构造函数一个数值
将会返回一个指定==字节数==的`ArrayBuffer`实例
```js
const buffer = new ArrayBuffer(16);
console.log(buffer.byteLength); // 16
```
==实现单独对每个字节的控制==
# 特点
+ 内存连续
+ 不能直接读写其内容
+ 需要通过 视图对象(Vied) 来操作
+ 创建后不能修改大小
	但可以通过`slice()`方法将其复制到一个新的实例
# 与c中的malloc的区别
+ malloc()在分配失败时会返回一个 null 指针
	ArrayBuffer 在分配失败时会抛出错误
+ malloc()可以利用虚拟内存，因此最大可分配尺寸只受可寻址系统内存限制
	ArrayBuffer分配的内存不能超过 Number.MAX_SAFE_INTEGER（$2^{53}-1$）字节
+ malloc()调用成功不会初始化实际的地址
	声明 ArrayBuffer 则会将==所有二进制位初始化为 0==
+ 通过 malloc()分配的堆内存除非调用 free()或程序退出，否则系统不能再使用
	==而通过声明ArrayBuffer 分配的堆内存可以被当成垃圾回收，不用手动释放==
