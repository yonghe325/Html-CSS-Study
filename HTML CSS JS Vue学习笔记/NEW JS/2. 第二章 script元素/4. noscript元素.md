`noscript`元素与 `div`标签类似
应在`noscript`元素内书写任何可以出现在`body`标签里的HTML元素
一般情况下, `noscript`元素里的内容不会被渲染
noscript元素 只适用于以下两种情况 : 
+ 浏览器不支持脚本
+ 浏览器关闭脚本功能

此时, noscript元素才会生效
才会正常渲染里面的标签
```js
<!DOCTYPE html>
<html>
	<head>
		<title>Example HTML Page</title>
		<script defer="defer" src="example1.js"></script>
		<script defer="defer" src="example2.js"></script>
	</head>
	<body>
		<noscript>
			<p>This page requires a JavaScript-enabled browser.</p>
		</noscript>
	</body>
</html>
```
