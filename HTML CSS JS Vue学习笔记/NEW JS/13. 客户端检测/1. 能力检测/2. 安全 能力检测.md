能力检测不能只检测特性是否存在
还要检测其是否能做出符合要求的行为
```js
// 不要这样做！错误的能力检测，只能检测到能力是否存在
function isSortable(object) {
	return !!object.sort;
}
```
只检测sort属性, 不代表sort就能对其排序
简单地测试到一个属性存在并不代表这个对象就可以排序
更好的方式是检测 sort 是不是函数
```js
// 好一些，检测 sort 是不是函数
function isSortable(object) {
	return typeof object.sort == "function";
}
```
==进行能力检测时应该尽量使用 typeof 操作符==
