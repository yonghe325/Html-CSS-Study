在 Web 发展早期, 所有用户的交互都会使整个页面刷新, 因为所有的运算都是在服务器执行
浏览器仅作为显示
所以当时的"前进","后退",的作用就是切换==状态==

## `pushState()`方法
`pushState()`方法**用于在不刷新页面的情况下向浏览器历史记录中添加新条目，同时改变地址栏的 URL**
这个方法接收 3 个参数：
一个 state 对象(可以是任意数据,数据大小有限制)
一个新状态的标题（可选的,大多数浏览器忽略该参数）
一个相对 URL
```js
let stateObject = {foo:"bar"};
history.pushState(stateObject, "My title", "baz.html");
```
`pushState()` 方法执行后，状态信息就会被推到历史记录中
浏览器地址栏也会改变以反映新的相对 URL
此时就可以通过"前进","后退"按钮切换状态
就可以为其绑定切换状态的 事件
