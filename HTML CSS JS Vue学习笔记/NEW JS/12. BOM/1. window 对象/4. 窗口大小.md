# 浏览器窗口自身大小
## 全局对象的`innerWidth`,`innerHeight`,`outerWidth` , `outerHeight`属性
`outerWidth` 和 `outerHeight` 返回浏览器窗口自身的大小
	不管是在最外层 window 上使用，还是在窗格\<frame>中使用
`innerWidth` 和 `innerHeight` 返回浏览器窗口中==页面视口==的大小
	即缩放的大小
	不包含浏览器边框和工具栏
## 全局对象的`document.documentElement.clientWidth` 和 `document.documentElement.clientHeight`属性
`document.documentElement.clientWidth` 和 `document.documentElement.clientHeight`属性
分别为页面视图的宽度 和 高度
浏览器窗口自身的精确尺寸不好确定，但可以确定页面视口的大小，如下所示：
```js
let pageWidth = window.innerWidth,
pageHeight = window.innerHeight;
if (typeof pageWidth != "number") {
	if (document.compatMode == "CSS1Compat"){
		//检测浏览器是否为标准模式
		pageWidth = document.documentElement.clientWidth;
		pageHeight = document.documentElement.clientHeight;
	} else {
		pageWidth = document.body.clientWidth;
		pageHeight = document.body.clientHeight;
	}
}
```
# 调整浏览器窗口
## `resignBy()` 方法 与 `resignTo()`方法
`resignBy()`接收 宽度 与 高度的缩放大小
`resignTO()`接收 新的宽度 与 新的高度
```js
// 缩放到 100×100
window.resizeTo(100, 100);

// 缩放到 200×150
window.resizeBy(100, 50);

// 缩放到 300×300
window.resizeTo(300, 300);
```
默认禁止 JavaScript 调整主窗口大小
除非窗口是通过 `window.open()` 打开的
