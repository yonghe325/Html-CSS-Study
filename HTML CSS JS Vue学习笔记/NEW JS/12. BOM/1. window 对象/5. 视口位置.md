# 查看滚动位置
浏览器窗口尺寸通常无法满足完整显示整个页面
为此用户可以通过滚动在有限的视口中查看文档
度量文档相对于视口滚动距离的属性有两对，返回相等的值
## `window.pageXoffset` 与 `window.scrollX`属性
返回x轴滚动的距离
## `window.pageYoffset` 与 `window.scrollY`属性
返回y轴方向滚动的值

# 滚动页面
```js
// 正常滚动
window.scrollTo({
	left: 100,
	top: 100,
	behavior: 'auto'
});
// 平滑滚动
window.scrollTo({
	left: 100,
	top: 100,
	behavior: 'smooth'
});
```
对于以下方法, 也可以传入一个 `ScrollToOptions` 字典
更精准的控制滚动页面
## `scroll()`
## `scrollTo()`
绝对位置
```js
// 滚动到页面左上角
window.scrollTo(0, 0);
// 滚动到距离屏幕左边及顶边各 100 像素的位置
window.scrollTo(100, 100);
```
## `scrollBy()`
相对于当前位置滚动的值
```js
// 相对于当前视口向下滚动 100 像素
window.scrollBy(0, 100);
// 相对于当前视口向右滚动 40 像素
window.scrollBy(40, 0);
```
