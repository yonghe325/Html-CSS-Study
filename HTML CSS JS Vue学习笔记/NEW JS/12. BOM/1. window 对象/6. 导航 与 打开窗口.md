# `window.open()`方法
```js
window.open("<要加载的URL>"[,"<目标窗口>","<特性字符串>",[<bool>]])
//bool值表示窗口是否打开
```
## 参数:
### `目标窗口` 
当`"<目标窗口>"`已经存在时, 会直接打开已有的窗口, 不会新建窗口
`"<目标窗口>"`也可以是特殊的窗口名
### `特性 字符串`
在`window.open()`方法中, `"<特性字符串>"`是一个逗号分隔的字符串,用于指定新窗口包含的特性
特性如下:

| 设置         | 值          | 说明                        |
| ---------- | ---------- | ------------------------- |
| fullscreen | "yes"或"no" | 表示新窗口是否最大化                |
| height     | 数值         | 新窗口高度, 值不能低于100           |
| left       | 数值         | 新窗口的x轴坐标, 不能为负数           |
| location   | "yes"或"no" | 是否显示地址栏, 默认值不确定           |
| Menubar    | "yes"或"no" | 是否显示菜单栏, 默认为"no"          |
| resizable  | "yes"或"no" | 是否可以通过拖动改变新窗口的大小, 默认为"no" |
| scrollbars | "yes"或"no" | 内容过长时是否滚动, 默认为"no"        |
| status     | "yes"或"no" | 是否显示状态栏, 默认值不确定           |
| toolbar    | "yes"或"no" | 是否显示工具栏, 默认为"no"          |
| top        | 数值         | 新窗口的y轴坐标, 不能为负数           |
| width      | 数值         | 新窗口的宽度, 值不能低于100          |
特性与值通过`=`连接, 不同特性通过`,`连接
## 返回值
返回一个对新窗口的引用, 类似于`window`对象
任何操作窗口的方法, 都适用于该返回值
该返回值有一个指针属性`opener`, 指向打开该新窗口的窗口
```js
let wroxWin = window.open("http://www.wrox.com/",
						"wroxWindow",
						"height=400,width=400,top=10,left=10,resizable=yes");
alert(wroxWin.opener === window); // true
```
>如果一个标签页打开了另一个
>而 window对象需要跟另一个标签页通信, 那么标签便不能运行在独立的进程中

可以手动将`opener`属性赋值为`null`
此时新窗口将与原来的窗口切断联系
运行在独立的进程中

# 弹窗屏蔽程序
大多数意料之外的弹窗都会被浏览器屏蔽
此时`open()`的返回值为null
且会抛出错误
```js
let blocked = false;
try {
	let wroxWin = window.open("http://www.wrox.com", "_blank");
	if (wroxWin == null){
		blocked = true;
	}
} catch (ex){
	blocked = true;
}
if (blocked){
	alert("The popup was blocked!");
}
```
