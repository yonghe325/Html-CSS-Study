>虽然 ECMAScript 把浏览器对象模型（BOM，Browser Object Model）描述为 JavaScript 的核心
>但实际上 BOM 是使用 JavaScript 开发 Web 应用程序的核心
>BOM 提供了与网页无关的浏览器功能对象。
>多年来，BOM 是在缺乏规范的背景下发展起来的，因此既充满乐趣又问题多多
>毕竟，浏览器开发商都按照自己的意愿来为它添砖加瓦
>最终，浏览器实现之间共通的部分成为了事实标准，为 Web 开发提供了浏览器间互操作的基础
>HTML5 规范中有一部分涵盖了 BOM 的主要内容
>因为 W3C 希望将JavaScript 在浏览器中最基础的部分标准化
