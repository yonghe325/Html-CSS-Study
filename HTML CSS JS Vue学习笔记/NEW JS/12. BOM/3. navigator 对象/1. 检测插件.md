检测浏览器是否安装了某个插件是开发中常见的需求
通过 `plugins` 数组来确定插件的信息
这个数组中的每一项都包含如下属性。
+ name：插件名称
+ description：插件介绍
+ filename：插件的文件名
+ length：由当前插件处理的 MIME 类型数量

```js
    function hasPlugin() {
      // 使用 for 循环（推荐）
      for (let i = 0; i < navigator.plugins.length; i++) {
        const pluginName = navigator.plugins[i].name.toLowerCase()
        console.log(pluginName)
      }
    }
    hasPlugin()
```
