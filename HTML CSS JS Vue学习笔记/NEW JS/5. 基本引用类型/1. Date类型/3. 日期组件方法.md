`getXxx()`：获取值
`setXxx()`：设置值
`getUTCXxx()`：获取UTC值
`setUTCXxx()`：设置UTC值
以下是所有Date类型的方法

| 方法分类      | 方法名                      | 本地时间 | UTC时间 | 作用         | 返回值范围         |
| --------- | ------------------------ | ---- | ----- | ---------- | ------------- |
| **获取时间戳** | `getTime()`              | ✅    | -     | 获取毫秒时间戳    | 0+            |
|           | `valueOf()`              | ✅    | -     | 同getTime() | 0+            |
| **设置时间戳** | `setTime(ms)`            | ✅    | -     | 设置毫秒时间戳    | Date对象        |
| **年份相关**  | `getFullYear()`          | ✅    | -     | 获取4位年份     | 如2024         |
|           | `getUTCFullYear()`       | -    | ✅     | 获取UTC年份    | 如2024         |
|           | `setFullYear(year)`      | ✅    | -     | 设置年份       | Date对象        |
|           | `setUTCFullYear(year)`   | -    | ✅     | 设置UTC年份    | Date对象        |
| **月份相关**  | `getMonth()`             | ✅    | -     | 获取月份       | 0(1月)-11(12月) |
|           | `getUTCMonth()`          | -    | ✅     | 获取UTC月份    | 0-11          |
|           | `setMonth(month)`        | ✅    | -     | 设置月份       | Date对象        |
|           | `setUTCMonth(month)`     | -    | ✅     | 设置UTC月份    | Date对象        |
| **日期相关**  | `getDate()`              | ✅    | -     | 获取月份中的日    | 1-31          |
|           | `getUTCDate()`           | -    | ✅     | 获取UTC日     | 1-31          |
|           | `setDate(date)`          | ✅    | -     | 设置日        | Date对象        |
|           | `setUTCDate(date)`       | -    | ✅     | 设置UTC日     | Date对象        |
| **星期相关**  | `getDay()`               | ✅    | -     | 获取星期几      | 0(日)-6(六)     |
|           | `getUTCDay()`            | -    | ✅     | 获取UTC星期几   | 0-6           |
| **小时相关**  | `getHours()`             | ✅    | -     | 获取小时       | 0-23          |
|           | `getUTCHours()`          | -    | ✅     | 获取UTC小时    | 0-23          |
|           | `setHours(hours)`        | ✅    | -     | 设置小时       | Date对象        |
|           | `setUTCHours(hours)`     | -    | ✅     | 设置UTC小时    | Date对象        |
| **分钟相关**  | `getMinutes()`           | ✅    | -     | 获取分钟       | 0-59          |
|           | `getUTCMinutes()`        | -    | ✅     | 获取UTC分钟    | 0-59          |
|           | `setMinutes(min)`        | ✅    | -     | 设置分钟       | Date对象        |
|           | `setUTCMinutes(min)`     | -    | ✅     | 设置UTC分钟    | Date对象        |
| **秒钟相关**  | `getSeconds()`           | ✅    | -     | 获取秒钟       | 0-59          |
|           | `getUTCSeconds()`        | -    | ✅     | 获取UTC秒钟    | 0-59          |
|           | `setSeconds(sec)`        | ✅    | -     | 设置秒钟       | Date对象        |
|           | `setUTCSeconds(sec)`     | -    | ✅     | 设置UTC秒钟    | Date对象        |
| **毫秒相关**  | `getMilliseconds()`      | ✅    | -     | 获取毫秒       | 0-999         |
|           | `getUTCMilliseconds()`   | -    | ✅     | 获取UTC毫秒    | 0-999         |
|           | `setMilliseconds(ms)`    | ✅    | -     | 设置毫秒       | Date对象        |
|           | `setUTCMilliseconds(ms)` | -    | ✅     | 设置UTC毫秒    | Date对象        |
| **时区相关**  | `getTimezoneOffset()`    | ✅    | -     | 获取时区偏移（分钟） | -720到+840     |
# 年份方法
|方法|本地/UTC|参数|说明|示例|
|---|---|---|---|---|
|`getFullYear()`|本地|无|4位年份|`2024`|
|`getUTCFullYear()`|UTC|无|UTC 4位年份|`2024`|
|`setFullYear()`|本地|`year`, `month?`, `date?`|设置年份，可选月日|`date.setFullYear(2025, 5, 15)`|
|`setUTCFullYear()`|UTC|`year`, `month?`, `date?`|设置UTC年份|`date.setUTCFullYear(2025)`|
# 月份方法
|方法|本地/UTC|参数|返回值/效果|注意|
|---|---|---|---|---|
|`getMonth()`|本地|无|0-11|0=1月，11=12月|
|`getUTCMonth()`|UTC|无|0-11|同上|
|`setMonth()`|本地|`month`, `date?`|设置月|month>11会增加年份|
|`setUTCMonth()`|UTC|`month`, `date?`|设置UTC月|同上|
# 日期方法
|方法|本地/UTC|参数|返回值/效果|注意|
|---|---|---|---|---|
|`getDate()`|本地|无|1-31|月份中的第几天|
|`getUTCDate()`|UTC|无|1-31|同上|
|`setDate()`|本地|`date`|设置日|date>月天数会增加月份|
|`setUTCDate()`|UTC|`date`|设置UTC日|同上|
# 星期方法
| 方法            | 本地/UTC | 返回值 | 对应关系             | 示例      |
| ------------- | ------ | --- | ---------------- | ------- |
| `getDay()`    | 本地     | 0-6 | 0=周日，1=周一...6=周六 | `3`（周三） |
| `getUTCDay()` | UTC    | 0-6 | 同上               | `2`（周二） |
# 时间方法
|方法类别|本地方法|UTC方法|参数|范围|溢出处理|
|---|---|---|---|---|---|
|**小时**|`getHours()`|`getUTCHours()`|无|0-23|-|
||`setHours(h)`|`setUTCHours(h)`|`h`, `min?`, `sec?`, `ms?`|-|h>23会增加天数|
|**分钟**|`getMinutes()`|`getUTCMinutes()`|无|0-59|-|
||`setMinutes(m)`|`setUTCMinutes(m)`|`m`, `sec?`, `ms?`|-|m>59会增加小时|
|**秒钟**|`getSeconds()`|`getUTCSeconds()`|无|0-59|-|
||`setSeconds(s)`|`setUTCSeconds(s)`|`s`, `ms?`|-|s>59会增加分钟|
|**毫秒**|`getMilliseconds()`|`getUTCMilliseconds()`|无|0-999|-|
||`setMilliseconds(ms)`|`setUTCMilliseconds(ms)`|`ms`|-|ms>999会增加秒钟|
