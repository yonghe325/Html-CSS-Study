# 引用类型 与 类 的区别
引用值 (或者对象) 是某个特定 引用类型 的实例
就像是 `1234` 这一串数字(值) 是 数值类型(类型) 一样
引用类型也被称为==对象定义==
在ECMAScript中, 所有数据结构都是引用类型, 甚至包括函数也是

引用类型 和 类(c++) 不一样
ECMAScript中的引用类型 缺少 其他面向对象语言中所具备的某些基本结构
包括 类和接口

与之相对的, ECMAScript使用了==原型==等机制, 用来代替传统的 类的机制
虽然ES6中添加了`class`这一关键字, 但本质上还是使用了原型等机制
所以`class`是语法糖, 并不是真正的类
# 实例 + 构造函数
对象其实就是某个特定引用类型的实例
新对象通过`new`关键字 + *构造函数* 来创建
所以==构造函数就是用来创建新对象的函数==
