# 简介
## `toString()`将对象转换为字符串表示
```js
// 作用：将对象转换为字符串表示
// 设计给：开发者、调试、日志记录

const date = new Date();
console.log(date.toString());
// "Mon Jan 15 2024 18:30:00 GMT+0800 (中国标准时间)"
// ↑ 开发者容易理解的格式

const arr = [1, 2, 3];
console.log(arr.toString());  // "1,2,3"

const num = 123;
console.log(num.toString());  // "123"

const obj = { x: 1 };
console.log(obj.toString());  // "[object Object]" ← 默认的没啥用
```
## `toLocaleString()` 根据用户地区和文化习惯将字符串表示格式化
```js
// 作用：根据用户地区和文化习惯格式化
// 设计给：最终用户、国际化应用

const date = new Date();
console.log(date.toLocaleString('zh-CN'));
// "2024/1/15 18:30:00" ← 中文用户习惯的格式

console.log(date.toLocaleString('en-US'));
// "1/15/2024, 6:30:00 PM" ← 美国用户习惯的格式

const num = 1234567.89;
console.log(num.toLocaleString('zh-CN'));
// "1,234,567.89"

console.log(num.toLocaleString('de-DE'));
// "1.234.567,89" ← 德国用逗号作小数点
```
## `valueOf()`获取对象的原始值
```js
// 作用：获取对象的原始值
// 设计给：数值运算、比较操作、类型转换

const date = new Date();
console.log(date.valueOf());
// 1705314600000 ← 时间戳（机器友好的数值）

const numObj = new Number(42);
console.log(numObj.valueOf());  // 42 ← 原始数字

// 实际应用：日期比较和计算
const date1 = new Date('2024-01-01');
const date2 = new Date('2024-01-02');

console.log(date2 > date1);  // true（比较时间戳）
console.log(date2 - date1);  // 86400000（一天的毫秒数）
```
# 重写
对于引用类型/包装类型来说, 默认的这些方式可能没用, 所以会对其进行重写, 如下 : 

| 类型           | toString()     | toLocaleString() | valueOf()  |
| ------------ | -------------- | ---------------- | ---------- |
| **Date**     | 可读日期字符串        | 本地化日期格式          | 时间戳        |
| **Array**    | 逗号分隔值          | 本地化逗号分隔          | 数组本身       |
| **Number**   | 数字字符串          | 本地化数字格式          | 数字==值==    |
| **Boolean**  | "true"/"false" | "true"/"false"   | true/false |
| **Function** | 函数源代码          | 函数源代码            | 函数本身       |
| **RegExp**   | pattern字面量     | pattern字面量       | 正则表达式本身    |
| **String**   | 原字符串           | 原字符串             | 原字符串       |
