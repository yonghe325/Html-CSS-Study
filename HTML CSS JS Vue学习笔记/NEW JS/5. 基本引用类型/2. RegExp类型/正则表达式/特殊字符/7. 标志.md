标志改变表达式的输出
标志决定表达式是否将文本视作单独的行处理，是否区分大小写，或者是否查找所有匹配项
# 全局标志 g
全局标志使表达式选中所有匹配项, 如果不启用全局标志
那么表达式只会匹配第一个匹配项
# 多行标志 m
正则表达式将所有文本视作一行
但如果使用了多行标志
它就会单独处理每一行
常配合全局标志g使用
# 单行模式 s
# 忽略大小写标记 i
使大小写不在敏感
# 内联标志 i
将标志放到正则表达式的内部
```js
(?i)python
//表示忽略p的大小写
```
# 粘附模式 y
强制正则表达式**必须从`lastIndex`开始匹配**
而不是在字符串中搜索
