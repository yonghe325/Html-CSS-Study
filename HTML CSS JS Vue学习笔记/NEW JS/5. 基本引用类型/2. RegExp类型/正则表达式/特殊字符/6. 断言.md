# 零宽断言
如果我们希望正在写的词语出现在另一个词语之前或之后
就需要使用零宽断言
## `(?=)`正向先行断言
```js
Date: 4 Aug 3PM

/\d+(?=PM)/
//匹配到3
//因为3后面接着PM
```
只匹配==后面==接着`(?=)`的字符
## `(?!)`负向先行断言
匹配后面不跟着`(?!)`的字符
就是`(?=)`的取反
## `(?<=)`正向后行断言
匹配==前面==有`(?<=)`的字符
```js
Product Code: 1064 Price: $5
/(?<=\$)\d+/
//匹配前面有$的数字
```
部分浏览器不支持
## `(?<!)`负向后行断言
即正向后行断言的取反
