# `()`分组
## `()` 分组
将所有匹配到的字符串进行分组
分组是相对于当前匹配来说的
例如
```js
[hah] [dad] bad dab [gag] gab
/(\w)a\1/g
//\1引用会引用当前匹配的\w
//在hah中, \w匹配h, 引用也会引用h
//在dad中, \w匹配d, 引用也会引用d
```
## `\<number>`引用分组
number表示正则表达式中的第number个分组
```js
\(ha)-\1,(haa)-\2\g
```
此处的`\1`引用的的是`(ha)`分组
`\2`引用的的是`(haa)`分组
## `(?:)` 非捕获分组
在引用时忽略该分组,防止其被`\<n>`捕获
```js
ha-ha,haa-haa
/(?:ha)-ha,(haa)-\1/g
//此处的\1指向(haa), 因为(?:ha)被屏蔽
```
# ` | `分支
## ` | `分支语法
` | `允许一个表达式包含多个不同的分支
允许表达式有多个匹配
类似于`[]`字符集, 但是`|`能匹配多个字符串
使用`()`划定级别
```js
/(cat|dog) food/
//cat food | dog food
/cat|dog food/
//cat | dog food
```
