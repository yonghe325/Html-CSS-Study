Number 是对应数值的引用类型
要创建一个 Number 对象，就使用 Number 构造函数并传入一个数值，如下例所示：
```js
let numberObject = new Number(10);
```
# 方法 与 继承方法的重写
## 重写
![[0.1 toLocaleString()、toString()和 valueOf() 与 重写#重写]]
```js
let num = 10;
console.log(num.toString()); // "10"
console.log(num.toString(2)); // "1010"
console.log(num.toString(8)); // "12"
console.log(num.toString(10)); // "10"
console.log(num.toString(16)); // "a"
```
## 其他方法
### `toFixed()`
传入一个整数(通常是0~20), 代表要保留的小数点后的数位
返回包含指定小数点位数的数值字符串
空位用0填充
```js
let num = 10;
console.log(num.toFixed(2)); // "10.00"
```
高位四舍五入
```js
let num = 10.005;
console.log(num.toFixed(2)); // "10.01"
```
### `toExponential()`
传入一个整数(通常是0~20), 代表要保留的小数点后的数位
返回以科学记数法(也称为指数记数法)表示的数值字符串
```js
let num = 10;
console.log(num.toExponential(1)); // "1.0e+1"
```
### `toPrecision()`
传入一个整数(通常是0~20), 代表要保留的数位(==不是小数点后了==)
==会根据实际情况==, 返回
指定数位的数值字符串 
==或者== 
以科学记数法(也称为指数记数法)表示的数值字符串
```js
let num = 99;
console.log(num.toPrecision(1)); // "1e+2"
console.log(num.toPrecision(2)); // "99"
console.log(num.toPrecision(3)); // "99.0"
```
## `Number.isInteger()`方法与安全整数
### `Number.isInteger()`
用于辨别一个数值是否保存为整数
==如果小数点后全是0, 也认为是整数==
```js
console.log(Number.isInteger(1)); // true
console.log(Number.isInteger(1.00)); // true
console.log(Number.isInteger(1.01)); // false
```
### 安全整数
>由于JavaScript使用 IEEE 754 的规范
而IEEE 754 数值格式有一个特殊的数值范围
==在这个范围内二进制值可以表示一个整数值==
这个数值范围从 
`Number.MIN_SAFE_INTEGER`($-2^{53}+1$)
到
`Number.MAX_SAFE_INTEGER`($2^{53}-1$)
对超出这个范围的数值, 即使尝试保存为整数
IEEE 754 编码格式也意味着二进制值可能会表示一个完全不同的数值
为了鉴别整数是否在这个范围内，可以使用 `Number.isSafeInteger()`方法
```js
console.log(Number.isSafeInteger(-1 * (2 ** 53))); // false
console.log(Number.isSafeInteger(-1 * (2 ** 53) + 1)); // true

console.log(Number.isSafeInteger(2 ** 53)); // false
console.log(Number.isSafeInteger((2 ** 53) - 1)); // true
```
