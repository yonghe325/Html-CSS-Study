CSS 样式在HTML 页面中可以通过两个元素加载
\<link>元素用于包含CSS 外部文件
而\<style>元素用于添加嵌入样式
与动态脚本类似，==动态样式也是页面初始加载时并不存在，而是在之后才添加到页面中的==
# 动态调整link元素
来看下面这个典型的\<link>元素：
```js
<link rel="stylesheet" type="text/css" href="styles .css">
```
这个元素很容易使用 DOM 编程创建出来：
```js
let link = document .createElement("link");
link.rel = "stylesheet";
link.type = "text/css";
link.href = "styles .css";
let head = document.getElementsByTagName("head")[0];
head.appendChild(link);
```
以上代码在所有主流浏览器中都能正常运行。注意应该把\<link>元素添加\<head>元素而不是\<body>元素
这样才能保证所有浏览器都能正常运行
这个过程可以抽象为以下通用函数：
```js
function loadStyles(url){
	let link = document .createElement("link");
	link .rel = "stylesheet";
	link .type = "text/css";
	link .href = url;
	let head = document .getElementsByTagName("head")[0];
	head .appendChild(link);
}
```
然后就可以这样调用这个 `loadStyles()` 函数了：
```js
loadStyles("styles.css");
```
==通过外部文件加载样式是一个异步过程==
因此，样式的加载和正执行的 JavaScript 代码并没有先后顺序
一般来说，也没有必要知道样式什么时候加载完成
# 脚本内部调整style元素
使用\<script>元素包含嵌入的 CSS 规则，例如：
```js
<style type="text/css">
	body {
		background-color: red;
	}
</style>
```
逻辑上，下列 DOM 代码会有同样的效果
```js
let style = document.createElement("style");
style.type = "text/css";
style.appendChild(document.createTextNode("body{background-color:red}"));
let head = document.getElementsByTagName("head")[0];
head.appendChild(style);
```
IE可能不允许使用该方式, 所以对于IE内核, 需要使用另一种方式
```js
function loadStyleString(css){
	let style = document .createElement("style");
	style.type = "text/css";
	try{
		style.appendChild(document.createTextNode(css));
	} catch (ex){
		style.styleSheet.cssText = css;
		// style.styleSheet.cssText属性
	}
	let head = document.getElementsByTagName("head")[0];
	// 
	head.appendChild(style);
}
```
