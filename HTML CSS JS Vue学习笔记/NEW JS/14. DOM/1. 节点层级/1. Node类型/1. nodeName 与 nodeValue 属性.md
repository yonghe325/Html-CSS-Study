`nodeName` 与 `nodeValue` 保存着有关节点的信息
+ `nodeName` 为只读属性
	+ 对于元素节点(`Node.ELEMENT_NODE`)
		`nodeName`返回大写的标签名
	+ 对于非元素节点
		`nodeName`返回以`#`开头的类型字符串
+ `nodeValue`==为可读写属性==
	+ `nodeValue`根据节点类型, 返回不同的值
		+ 元素节点/文档节点返回`null`
		+ 文本节点返回文本内容
		+ 属性节点返回属性值
---
这两个属性的值完全取决于节点类型
在使用这两个属性前，==最好先检测节点类型==，如下所示：
```js
if (someNode.nodeType == 1){
	value = someNode.nodeName; // 会显示元素的标签名
}
