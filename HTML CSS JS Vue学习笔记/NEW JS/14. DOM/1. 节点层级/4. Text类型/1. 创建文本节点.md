# `document.createTextNode()`方法
`document.createTextNode()` 可以用来创建新文本节点，它接收一个参数，即要插入节点的文本
跟设置已有文本节点的值一样，这些要插入的文本也会应用 HTML 或 XML 编码，如下面的例子所示：
```js
let textNode = document .createTextNode("<strong>Hello</strong> world!");
```
创建新文本节点后，其 ownerDocument 属性会被设置为 document
但在把这个节点添加到文档树之前，我们不会在浏览器中看到它
以下代码创建了一个元素并给它添加了一段文本消息
```js
let element = document.createElement("div");
	// 创建元素节点
element.className = "message";
	// 修改类名
let textNode = document.createTextNode("Hello world!");
	// 创建文本节点
element.appendChild(textNode);
	// 将文本节点添加进新创建的元素节点
document.body.appendChild(element);
	// 将元素节点添加进DOM树中
```
---
一般来说一个元素只包含一个文本子节点
不过，也可以让元素包含多个文本子节点，如下面的例子所示：
```js
let element = document.createElement("div");
element.className = "message";
// 创建元素节点
let textNode = document.createTextNode("Hello world!");
element.appendChild(textNode);
// 添加第一个文本节点
let anotherTextNode = document.createTextNode("Yippee!");
element.appendChild(anotherTextNode);
//添加第二个文本节点
document.body.appendChild(element);
// 添加进DOM树中
```
在将一个文本节点作为另一个文本节点的同胞插入后，==两个文本节点的文本之间不会包含空格==
