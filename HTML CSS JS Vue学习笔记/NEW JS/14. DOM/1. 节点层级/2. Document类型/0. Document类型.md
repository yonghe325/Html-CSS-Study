Document 类型是 JavaScript 中表示文档节点的类型
在浏览器中，文档对象 document 是 HTMLDocument 的实例（HTMLDocument 继承 Document）
	==表示整个 HTML 页面==
document 是 window 对象的属性, 因此是一个全局对象
Document 类型的节点有以下特征
+ nodeType 等于 9；
+ nodeName 值为"#document"；
+ nodeValue 值为 null；
+ parentNode 值为 null；
+ ownerDocument 值为 null；
+ 子节点可以是 DocumentType（最多一个）、Element（最多一个）
	ProcessingInstruction或 Comment 类型
