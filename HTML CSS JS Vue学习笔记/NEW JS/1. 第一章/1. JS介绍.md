# 作用
1. 网页特效 : 根据监听行为作出反馈
2. 表单验证 : 针对表单数据的合法性进行判断
3. 数据交互 :   获取后台的数据, 渲染到前端
4. 服务端编程 : node.js

# 组成 
+ JavaScript
	+ ECMAScript : JavaScript语言基础
		规定了js基础语法的核心知识  : 变量,分支语句
	+ Web APIs 
		+ DOM : ==操作文档==，比如对页面元素进行移动、大小、添加删除等操作
		+ BOM : ==操作浏览器==，比如页面弹窗，检测圇口宽度、存储数据到浏览器等等

# 书写位置
+ 内部 : 直接写在html中,用script标签保住
	写在\</body>的==上方==,即body标签的底部
+ 外部-
+ 内联(行内) : vue中会经常使用
