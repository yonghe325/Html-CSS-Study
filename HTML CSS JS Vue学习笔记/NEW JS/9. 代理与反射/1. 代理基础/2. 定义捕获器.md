# 捕获器的概念
使用代理的主要目的 是为被代理对象定义 **捕获器**(trap)
捕获器 就是在处理程序对象中定义的“基本操作的拦截器”
每个处理程序对象可以包含零个或多个捕获器
每个捕获器都对应==一种基本操作==，可以直接或间接在代理对象上调用
每次在代理对象上调用这些基本操作时
	==一个操作可能由多个基本操作组合构成==
	捕获器会将这些基本操作全部拦截
	如果拦截一个非基本操作, 就需要手动设置拦截
代理可以在这些操作传播到目标对象之前先调用捕获器,函数
从而拦截并修改相应的行为
	即阻止默认行为的发生
# 定义捕获器
定义一个`get`捕获器
```js
const target = {
	foo: 'bar'
};

const handler = {
// 捕获器在处理程序对象中以方法名为键
	get() {
		return 'handler override';
	}
};
const proxy = new Proxy(target, handler);
```
当通过代理对象调用属性时, 就会被`get`捕获器拦截
不会返回原本的属性, 只会执行`get`捕获器中的代码
