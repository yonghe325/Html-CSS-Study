捕获处理程序的行为必须遵循 **“捕获器不变式”**（trap invariant）
	所有捕获器的捕获器不变式
	转到[[0. Proxy 捕获器完整参考表]]
==捕获器不变式因方法不同而异==, ==会防止捕获器定义出现过于反常的行为==
比如，如果目标对象有一个不可配置且不可写的数据属性
那么在捕获器返回一个与该属性不同的值时
```js
const target = {};
Object.defineProperty(target, 'foo', {
	configurable: false,
	writable: false,
	value: 'bar'
});
const handler = {
	get() {
		return 'qux';
	}
};
const proxy = new Proxy(target, handler);
console.log(proxy.foo);
	//使用代理访问foo属性
	//foo属性的值为bar
	//但捕获器捕获后返回qux
	//抛出异常
// TypeError
```
