`get()`捕获器会在获取属性值的操作中被调用
对应额度反射API为`Reflect.get()`
```js
get(target,property,receiver)
--------------------------------------
const myTarget = {};
const proxy = new Proxy(myTarget, {
	get(target, property, receiver) {
		console.log('get()');
		return Reflect.get(...arguments)
	}
});
proxy.foo;
// get()
```
## 返回值
只要符合[[4. 捕获器不变式|捕获器不变式]],可以返回任何值
## 可以被拦截的操作
1. `proxy.property`
2. `proxy[property]`
3. `Object.create(proxy)[property]`
	`Object.create(proxy)`以代理对象为原型创建对象
	新创建的对象也是被`proxy`代理对象的代理
	所以会被`get`捕获
 4. `Reflect.get(proxy, property, receiver)`
	主要用于嵌套代理中
## 参数
+ target：目标对象。
+ property：引用的目标对象上的==字符串==/==符号==键属性
+ receiver：代理对象 或 继承代理对象的对象(外层嵌套)
## 捕获器不变式
+ 如果`target.property`
	`[[configurable]]: false`且`[[writable]]: false`
	则处理程序返回的值必须与 `target.property`的`value` 匹配
+ 如果 `target.property` 
	`[[configurable]]: false`且`[[Get]]: undefined`
	处理程序的返回值也必须是 `undefined`
