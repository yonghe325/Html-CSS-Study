`getOwnPropertyDescriptor()`捕获器
会在 `Object.getOwnPropertyDescriptor()`中被调用
	关于`Object.getOwnPropertyDescriptor()`
	转到[[3. 读取属性的特性#`Object.getOwnPropertyDescriptor()`方法|Object.getOwnPropertyDescriptor()]]
对应的反射 API 方法为 `Reflect.getOwnPropertyDescriptor()`
```js
const myTarget = {};
const proxy = new Proxy(myTarget, {
	getOwnPropertyDescriptor(target, property) {
		console.log('getOwnPropertyDescriptor()');
		return Reflect.getOwnPropertyDescriptor(...arguments)
	}
});
Object.getOwnPropertyDescriptor(proxy, 'foo');
// getOwnPropertyDescriptor()
```
## 返回值
当目标属性不存在时,必须返回关于这个属性的一个对象
当目标属性不存在时, 返回`Undefined`
## 可以拦截的操作
+ Object.getOwnPropertyDescriptor(proxy, property)
	不能拦截Object.getOwnPropertyDescriptor==s==(proxy, property)
+ Reflect.getOwnPropertyDescriptor(proxy, property)
## 参数
+ target：目标对象。
+ property：引用的目标对象上的字符串键属性。
## 捕获器不变式
+ 当`target.property`存在, 必须返回对象
	+ 且`[[configurable]]: false`, 返回一个表示该 属性存在的对象
		捕获器的返回对象中的`configurable`属性必须为`false`
		且其他属性必须与源对象属性中的特性的值一致
	+ 且`[[configurable]]: true`, 返回该 属性可配置的对象
		捕获器的返回对象中的`configurable`属性必须为`true`
		且其他属性必须与源对象属性中的特性的值一致
	+ 且`[[Extensible]]: false`, 须返回一个表示该属性存在的对象
+ 当`target.property`不存在
	+ 不能返回返回表示该属性可配置的对象
	+ 且`target.[[Extensible]]: false`, 返回`Undefined`表示该属性不存在
