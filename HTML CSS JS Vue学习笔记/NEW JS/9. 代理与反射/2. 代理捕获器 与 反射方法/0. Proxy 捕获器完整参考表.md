
| 捕获器                            | 参数                                  | 拦截操作                                                                                                | 返回值                           | 捕获器不变式                                                        |
| ------------------------------ | ----------------------------------- | --------------------------------------------------------------------------------------------------- | ----------------------------- | ------------------------------------------------------------- |
| **`get`**                      | `target, property, receiver`        | 读取属性：`obj.prop`、`obj['prop']`                                                                       | 任意值                           | 如果 `target.property` 是不可写且不可配置的数据属性，返回值必须等于 `target.property` |
| **`set`**                      | `target, property, value, receiver` | 设置属性：`obj.prop = value`、`obj['prop'] = value`                                                       | 布尔值（`true` 表示成功，`false` 表示失败） | 如果 `target.property` 是不可写且不可配置的数据属性，不能修改其值                    |
| **`has`**                      | `target, property`                  | `in` 操作符：`prop in obj`                                                                              | 布尔值（表示属性是否存在）                 | 如果 `target.property` 是不可配置的，必须返回 `true`（不能隐藏）                 |
| **`defineProperty`**           | `target, property, descriptor`      | `Object.defineProperty()`                                                                           | 布尔值（表示是否定义成功）                 | 如果 `target` 不可扩展，不能添加新属性；如果属性不可配置，不能修改其描述符                    |
| **`getOwnPropertyDescriptor`** | `target, property`                  | `Object.getOwnPropertyDescriptor()`                                                                 | 属性描述符对象或 `undefined`          | 如果属性存在且不可配置，返回的描述符必须包含 `configurable: false`                  |
| **`deleteProperty`**           | `target, property`                  | `delete` 操作符：`delete obj.prop`                                                                      | 布尔值（表示是否删除成功）                 | 如果 `target.property` 是不可配置的，不能删除，必须返回 `false`                 |
| **`ownKeys`**                  | `target`                            | `Object.keys()`、`Object.getOwnPropertyNames()`、`Object.getOwnPropertySymbols()`、`Reflect.ownKeys()` | 可枚举对象（属性名数组）                  | 必须包含 `target` 的所有不可配置的自有属性；如果 `target` 不可扩展，返回值必须恰好包含所有自有属性   |
| **`getPrototypeOf`**           | `target`                            | `Object.getPrototypeOf()`、`__proto__`、`Object.prototype.isPrototypeOf()`                            | 对象或 `null`                    | 如果 `target` 不可扩展，返回值必须等于 `Object.getPrototypeOf(target)`      |
| **`setPrototypeOf`**           | `target, prototype`                 | `Object.setPrototypeOf()`                                                                           | 布尔值（表示是否设置成功）                 | 如果 `target` 不可扩展，不能修改原型，必须返回 `false`                          |
| **`isExtensible`**             | `target`                            | `Object.isExtensible()`                                                                             | 布尔值                           | 如果 `target` 不可扩展，必须返回 `false`                                 |
| **`preventExtensions`**        | `target`                            | `Object.preventExtensions()`                                                                        | 布尔值                           | 如果 `Object.isExtensible(target)` 已经是 `false`，必须返回 `true`      |
| **`apply`**                    | `target, thisArg, argumentsList`    | 函数调用：`func()`、`func.apply()`、`func.call()`                                                          | 任意值                           | `target` 必须是可调用的（函数对象）                                        |
| **`construct`**                | `target, argumentsList, newTarget`  | `new` 操作符：`new Constructor()`                                                                       | 对象                            | `target` 必须是可构造的；返回值必须是对象                                     |
