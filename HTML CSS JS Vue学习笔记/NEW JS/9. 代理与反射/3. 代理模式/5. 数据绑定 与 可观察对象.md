通过代理可以把运行时中原本不相关的部分联系到一起
这样就可以实现各种模式，从而让不同的代码互操作
比如，可以将被代理的类绑定到一个全局实例集合，让所有创建的实例都被添加到这个集合中
```js
const userList = [];
class User {
	constructor(name) {
		this.name_ = name;
	}
}
const proxy = new Proxy(User, {
	construct() {
		const newUser = Reflect.construct(...arguments);
		userList.push(newUser);
		return newUser;
	}
});
new proxy('John');
new proxy('Jacob');
new proxy('Jingleheimerschmidt');
console.log(userList); // [User {}, User {}, User{}]
```
另外，还可以把集合绑定到一个事件分派程序，每次插入新实例时都会发送消息：
```js
const userList = [];
function emit(newValue) {
	console.log(newValue);
}
const proxy = new Proxy(userList, {
	set(target, property, value, receiver) {
		const result = Reflect.set(...arguments);
		if (result) {
			emit(Reflect.get(target, property, receiver));
		}
			return result;
		}
	}
});
proxy.push('John');
proxy.push('Jacob');
```

所以在这一段代码中
push分两步操作:
1. 将userList的Length设为1
	这一步被set捕获, target为userlist, property为length,value为1,receiver为代理对象
	设置成功后, result为反射APIset的返回值true
	然后进入if,调用emit函数
	传入Reflect.get(target, property, receiver)返回值,
		Reflect.get()内部参数中
		target为userlist,property为lenght,receiver为代理对象
		Reflect.get()的到target.property,即为userlist.length
		此时length为1, 因为前面反射APIset已经将长度设为1
	返回值为 1 
	然后emit打印返回值'1'
2. 将值放入下标0
	这一步也被set捕获, target为userlist, property为???,value为'John',receiver为代理对象
	即将userlist下标为0的位置的值设为'John'
	设置成功后, result为反射APIset的返回值true
	然后进入if,调用emit函数
		传入Reflect.get(target, property, receiver)返回值,
		Reflect.get()内部参数中
			target为userlist,property为???,receiver为代理对象
			Reflect.get()的到target.property,即为userlist[0]
		返回值为 'John'
	然后emit打印返回值'john'
