### 1. 核心 HTML/DOM 事件（最常用）
这些是操作网页交互和布局的基础事件。
- **文档/窗口周期**：`DOMContentLoaded`, `load`, `beforeunload`, `unload`, `error`, `abort`
- **网络/连接状态**：`online`, `offline`, `pageshow`, `pagehide`
- **表单/控件**：`submit`, `reset`, `change`, `input`, `invalid`
- **视图/滚动**：`resize`, `scroll`
- **焦点**：`focus`, `blur`, `focusin`, `focusout`
- **资源加载**：`loadstart`, `progress`, `loadend`, `timeout`
- **历史/路由**：`hashchange`, `popstate`
- **存储**：`storage`
### 2. 设备与硬件事件（移动端/特殊设备）
这些事件与设备的物理传感器或硬件状态有关。
- **方向/运动**：`devicemotion`, `deviceorientation`, `compassneedscalibration`（罗盘校准）
- **环境光/接近**：`devicelight`（环境光）, `deviceproximity`, `userproximity`（接近传感器）
- **电池**：`chargingchange`, `levelchange`, `chargingtimechange`, `dischargingtimechange`
- **网络信息**：`change` (Network Information API)
- **屏幕方向**：`change` (Screen Orientation API)
- **全屏**：`fullscreenchange`, `fullscreenerror`
### 3. 剪贴板事件
- `beforecopy`, `copy`, `cut`, `paste`（包含对应的 `before` 变体）。
### 4. 拖放 API 事件
- `dragstart`, `drag`, `dragend`（拖拽源元素）
- `dragenter`, `dragover`, `dragleave`, `drop`（拖拽目标元素）
### 5. 媒体与资源事件（音视频、Canvas等）
- **音视频周期**：`play`, `playing`, `pause`, `ended`, `waiting`, `seeking`, `volumechange`, `ratechange`, `canplay`, `canplaythrough`
- **音视频队列/加载**：`loadstart`, `progress`, `suspend`, `emptied`, `stalled`
- **文本轨道**：`enter`, `exit`, `cuechange` (WebVTT)
### 6. 进阶 API 事件
这些事件用于更复杂的 Web 应用功能。
- **网络/通信**：
    - **WebSocket**: `open`, `message`, `close`, `error`
    - **Server Sent Events**: `open`, `message`, `error`
    - **频道消息**: `message` (Channel Messaging, Broadcast Channel)
    - **跨应用通信**: `message` (Inter-App Connection)
- **数据/存储**：
    - **IndexedDB**: `success`, `error`, `complete`, `abort`, `versionchange`, `blocked`, `upgradeneeded`
    - **文件**: `load`, `error`, `progress`, `abort` (File API, File System API)
- **离线与服务 Worker**：
    - **App Cache**: `checking`, `downloading`, `updateready`, `cached`, `noupdate`, `obsolete`（已部分废弃）
    - **Service Workers**: `install`, `activate`, `fetch`, `message`, `updatefound`, `controllerchange`
- **高级图形与 UI**：
    - **动画**: `animationstart`, `animationend`, `animationiteration`
    - **过渡**: `transitionend`
    - **WebGL**: `webglcontextlost`, `webglcontextrestored`, `webglcontextcreationerror`
    - **组件/阴影DOM**: `slotchange`
- **支付与权限**：
    - **Payment Request**: `shippingaddresschange`, `shippingoptionchange`
### 7. 遗留或已废弃事件（建议了解但不优先使用）
这些事件可能兼容性不佳或被现代标准替代。
- **DOM变异事件 (Mutation Events)**：`DOMSubtreeModified`, `DOMNodeInserted`, `DOMNodeRemoved` 等（效率低，已被 `MutationObserver` 替代）。
- **设备/特定API**：
    - **App Cache**: 上述列表已注明。
    - **触摸 (Touch Events)**：`touchstart`, `touchmove`, `touchend`, `touchcancel`（移动端核心，但现代开发可用 `Pointer Events` 统一）。
    - **指针 (Pointer Events)**：`pointerdown`, `pointermove`, `pointerup` 等（现代指针事件统一方案）。
    - **SVG 特有**：`SVGLoad`, `SVGError`, `SVGZoom` 等。
