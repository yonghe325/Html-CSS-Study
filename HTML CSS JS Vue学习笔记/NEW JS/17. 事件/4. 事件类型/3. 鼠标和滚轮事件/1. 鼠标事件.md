# 鼠标事件
DOM3 Events定义了 9 种鼠标事件: 
+ 冒泡
	+ click: ==单击==
		在用户单击鼠标主键（通常是左键）或按键盘回车键时触发
		这主要是基于无障碍的考虑，让键盘和鼠标都可以触发 onclick 事件处理程序。
	+ dblclick: ==双击==
		在用户双击鼠标主键（通常是左键）时触发
	+ mousedown: ==按下==
		在用户按下任意鼠标键时触发
	+ mouseup: ==释放==
		在用户释放鼠标键时触发
	+ mousemove: ==移动==
		在鼠标光标在元素上移动时反复触发
	+ mouseout: ==切换==
		在用户把鼠标光标从一个元素移到另一个元素上时触发
		移到的元素可以是原始元素的外部元素，也可以是原始元素的子元素
	+ mouseover: ==进入==
		在用户把鼠标光标从元素外部移到元素内部时触发
+ 不冒泡
	+ mouseenter: ==进入==
		在用户把鼠标光标从元素外部移到元素内部时触发
		这个事件==不冒泡==，也不会在光标经过后代元素时触发
		mouseenter 事件不是在 DOM2 Events 中定义的，而是 DOM3 Events中新增的事件
	+ mouseleave: ==离开==
		在用户把鼠标光标从元素内部移到元素外部时触发
		这个事件==不冒泡==，也不会在光标经过后代元素时触发
		mouseleave 事件不是在 DOM2 Events 中定义的，而是 DOM3 Events中新增的事件

除此之外还有[[7. mousewheel 事件|mousewheel 事件]]

---
鼠标事件之间存在关系, 因此取消鼠标事件的默认行为也会影响其他事件
例如 click, dbclick, mousedown,mouseup这四个事件永远都会按照以下顺序触发:
1. mousedown
2. mouseup
3. click
4. mousedown
5. mouseup
6. click
7. dblclick

如果取消mousedown 或 mouseup的默认事件, 则click 与 dbclick也不会触发
mousedown 和 mouseup 则不会受其他事件影响
