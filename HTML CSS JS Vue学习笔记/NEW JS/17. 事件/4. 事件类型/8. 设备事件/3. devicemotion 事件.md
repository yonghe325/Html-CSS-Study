DeviceOrientationEvent 规范也定义了 devicemotion 事件
这个事件用于提示设备实际上在移动，而不仅仅是改变了朝向
例如，devicemotion 事件可以用来确定设备正在掉落或者正拿在一个行走的人手里

当 devicemotion 事件触发时，event 对象中包含如下额外的属性
+ acceleration : 对象，包含 x、y 和 z 属性
	反映不考虑重力情况下各个维度的加速信息
+ accelerationIncludingGravity : 对象，包含 x、y 和 z 属性
	反映各个维度的加速信息，包含 z 轴自然重力加速度。
+ interval : 毫秒，距离下次触发 devicemotion 事件的时间
	此值在事件之间应为常量。
+ rotationRate : 对象，包含 alpha、beta 和 gamma 属性，表示设备朝向

如果无法提供 acceleration、accelerationIncludingGravity 和 rotationRate 信息, 则属性值为 null
```js
window.addEventListener("devicemotion", (event) => {
	let output = document.getElementById("output");
	if (event.rotationRate !== null) {
		output.innerHTML += `Alpha=${event.rotationRate.alpha}` +
			`Beta=${event.rotationRate.beta}` +
			`Gamma=${event.rotationRate.gamma}`;
	}
});
```
