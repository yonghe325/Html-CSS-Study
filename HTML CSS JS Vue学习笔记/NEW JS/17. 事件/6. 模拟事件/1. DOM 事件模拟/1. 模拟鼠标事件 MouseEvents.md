模拟鼠标事件需要先创建一个新的鼠标 event 对象，然后再使用必要的信息对其进行初始化
要创建鼠标 event 对象，可以调用 createEvent()方法并传入"MouseEvents"参数
这样就会返回一个 event 对象
这个对象有一个 `initMouseEvent()`方法，用于为新对象指定鼠标的特定信息
`initMouseEvent()`方法接收 15 个参数，分别对应鼠标事件会暴露的属性
这些参数列举如下:
1. `type`(字符串) : 要触发的事件类型，如"click"
2. `bubbles`(布尔值) : 表示事件是否冒泡
	为精确模拟鼠标事件，应该设置为 true
3. `cancelable`(布尔值) : 表示事件是否可以取消
	为精确模拟鼠标事件，应该设置为 true
4. `view`(AbstractView) : 与事件关联的视图
	基本上始终是 document.defaultView
5. `detail`(整数) : 关于事件的额外信息
	只被事件处理程序使用，通常为 0
6. `screenX`(整数) : 事件相对于屏幕的 x 坐标
7. `screenY`(整数) : 事件相对于屏幕的 y 坐标
8. `clientX`(整数) : 事件相对于视口的 x 坐标
9. `clientY`(整数) : 事件相对于视口的 y 坐标
10. `ctrlkey`(布尔值) : 表示是否按下了 Ctrl 键
	默认为 false
11. `altkey`(布尔值) : 表示是否按下了 Alt 键
	默认为 false
12. `shiftkey`(布尔值) : 表示是否按下了 Shift 键
	默认为 false
13. `metakey`(布尔值) : 表示是否按下了 Meta 键
	默认为 false
14. `button`(整数) : 表示按下了哪个按钮
	默认为 0
15. `relatedTarget`(对象) : 与事件相关的对象
	只在模拟 mouseover 和 mouseout 时使用
	其他事件为null

使用默认值模拟单击事件的例子:
```js
let btn = document.getElementById("myBtn");
// 创建 event 对象
let event = document.createEvent("MouseEvents");
// 初始化 event 对象
event.initMouseEvent("click", true, true, document.defaultView,0, 0, 0, 0, 0, false, false, false, false, 0, null);
// 触发事件
btn.dispatchEvent(event);
```
所有鼠标事件，包括 dblclick 都可以像这样在 DOM 合规的浏览器中模拟出来
