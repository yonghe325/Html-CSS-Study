DOM2 Style 规范也在 style 对象上定义了一些属性和方法
这些属性和方法提供了元素 style 属性的信息并支持修改，列举如下:
+ `cssText` : 包含在 style 属性中的 CSS 代码
+ `length` : 应用给元素的 CSS 属性*数量*
+ `parentRule` :表示 CSS 信息的 CSSRule 对象（下一节会讨论 CSSRule 类型）
+ `getPropertyCSSValue(propertyName)` : 返回包含 CSS 属性 [^1]`propertyName` 值的 ~~CSSValue对象~~（已废弃）
+ `getPropertyPriority(propertyName)` : 如果 CSS 属性 [^1]`propertyName` 使用了!important则返回"important"
	否则返回空字符串
+ `getPropertyValue(propertyName)` : 返回属性 [^1]`propertyName` 的字符串值
+ `item(index)` : 返回索引为 index 的 CSS 属性名
+ `removeProperty(propertyName)` : 从样式中删除 CSS 属性 [^1]`propertyName`
+ `setProperty(propertyName, value, priority)` : 设置 CSS 属性 [^1]`propertyName` 的值为 `value`
	priority 是"important" 或 空字符串
---
# `cssText`属性
在读取时, `cssText`会返回style属性css代码在浏览器内部的表示
在设置时, `cssText`==会以重写的方式重写整个style属性的值==
	意味着之前通过style 属性设置的属性都会丢失
比如，如果一个元素通过 style 属性设置了边框，而赋给 cssText属性的值不包含边框，则元素的边框会消失
下面的例子演示了 cssText 的使用：
```js
myDiv.style.cssText = "width: 25px; height: 100px; background-color: green";
console.log(myDiv.style.cssText);
```
设置 cssText 是一次性修改元素多个样式最快捷的方式，因为==所有变化会同时生效==

# `length`属性
length 属性是跟 `item()`方法一起配套迭代 CSS 属性用的
此时，==style 对象实际上变成了一个集合==
	此时位置为键, 对应位置样式的 `propertyName`为值 
也可以用中括号代替 `item()`取得相应位置的 CSS 属性名，如下所示：
```js
for (let i = 0, len = myDiv.style.length; i < len; i++) {
	console.log(myDiv.style[i]); // 或者用 myDiv.style.item(i)
}
```
使用中括号或者 `item()`都可以取得相应位置的 CSS 属性名（"background-color"，不是"backgroundColor"）
	即 `propertyName`
这个属性名可以传给 getPropertyValue()以取得属性的值
如下面的例子所示：
```js
let prop, value, i, len;
for (i = 0, len = myDiv.style.length; i < len; i++) {
	prop = myDiv.style[i]; // 或者用 myDiv.style.item(i)
	value = myDiv.style.getPropertyValue(prop);
	console.log(`prop: ${value}`);
}
```
# `getPropertyValue()` 方法 与 `getPropertyCSSValue()`方法 
`getPropertyValue()`返回 CSS 属性值的字符串表示
如果需要更多信息，则可以通过`getPropertyCSSValue()` 获取 CSSValue 对象
这个对象有两个属性：`cssText` 和 `cssValueType`
+ `cssText`的值与 getPropertyValue()方法返回的值一样；
+ `cssValueType`是一个数值常量，表示当前值的类型（0代表继承的值,1 代表原始值，2 代表列表，3 代表自定义值)
	样式的类型, 转到[[4. 样式的类型|样式的类型]]
```js
let prop, value, i, len;
for (i = 0, len = myDiv.style.length; i < len; i++) {
	prop = myDiv.style[i]; // alternately, myDiv.style.item(i)
		// 获取propertyName
	value = myDiv.style.getPropertyCSSValue(prop);
		// 获取 CSSValue 对象
	console.log(`prop: ${value.cssText} (${value.cssValueType})`);
		// cssText = propertyName, cssValueType 代表类型
}
```
## `getPropertyCSSValue()`方法的实现
元素的style属性并不包含 继承与来自外部的样式
但`getPropertyCSSValue()`方法可以获取来自外部 与 继承的元素
是因为`getPropertyCSSValue()`内部调用的是`window.getComputedStyle(element)`方法
	转到[[2. 计算样式]]
# removeProperty()方法
**恢复默认值**
`removeProperty()` 方法用于从元素样式中删除指定的 CSS 属性
使用这个方法删除属性意味着会应用该属性的默认（从其他样式表层叠继承的）样式
例如，可以像下面这样删除 style 属性中设置的 border 样式：
```js
myDiv.style.removeProperty("border");
```
在不确定给定 CSS 属性的默认值是什么的时候，可以使用这个方法
只要从 style 属性中删除，就可以使用默认值

[^1]: `propertyName`指的是以 ==连字符表示法css样式== 的属性字符串
