# contentDocument属性
在DOM2 中, 给HTMLIFrameElement类元素新增了一个属性`contentDocument`
这个属性包含代表子内嵌窗格中内容的document对象的指针
```js
let iframe = document.getElementById("myIframe");
let iframeDoc = iframe.contentDocument;
```
contentDocument属性 是 Document类型的实例, 拥有所有文档属性和方法
因此可以像使用HTML文档一样使用它
# contentWindow属性
返回相应窗格的Window对象, 找个对象有一个document属性
所有现代浏览器都支持 contentDocument 和 contentWindow 属性
