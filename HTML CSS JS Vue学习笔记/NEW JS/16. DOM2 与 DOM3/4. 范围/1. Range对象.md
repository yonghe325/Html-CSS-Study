DOM2 在 Document 类型上定义了一个 `createRange()`方法，暴露在 `document` 对象上
使用这个方法可以创建一个 DOM 范围对象(Range类型)，如下所示：
```js
let range = document.createRange();
```
与节点类似，这个新创建的范围对象是与创建它的文档关联的，不能在其他文档中使用
然后可以使用这个范围在后台选择文档特定的部分
创建范围并指定它的位置之后，可以对范围的内容执行一些操作，从而实现对底层 DOM 树更精细的控制
	==所以Range对象在创建时为空==
每个范围都是 Range 类型的实例，拥有相应的属性和方法
下面的属性提供了与范围在文档中位置相关的信息 : 
+ `startContainer` : 
	范围起点所在的节点（选区中第一个子节点的父节点）
+ `endContainer` : 
	范围终点所在的节点（选区中最后一个子节点的父节点）
+ `startOffset` : 
	范围起点在 `startContainer` 中的==偏移量==
	如果 `startContainer` 是文本节点、注释节点或 CData 区块节点
		则 `startOffset` 指范围起点之前跳过的字符数
	否则，表示范围中第一个节点的索引
+ `endOffset` : 
	范围起点在 `startContainer` 中的偏移量（与 startOffset 中偏移量的含义相同）
+ `commonAncestorContainer` : 
	文档中以`startContainer`和 `endContainer`为后代的最深的节点

这些属性会在范围被放到文档中特定位置时获得相应的值
	即只有设置之后, 才能获取相应的值
# 属性的辨析
范围可以在一个节点之前开始或一个节点之后开始
也可以在一个节点之前开始或一个节点桌结束
总结:
==`startOffset` 与 `endOffset`永远指向开始结束节点/字符之前==
`commonAncestorContainer`永远指向`startContainer` 与 `endContainer`的最低公共祖先
## 开始
### 当开始节点为元素节点时
+ 如果在一个节点之前开始, 则范围包括这个节点
	此时`startContainer`为开始节点的父结点
	`startOffset`为`startContainer`中, 开始节点的的索引
+ 如果在一个节点之后开始, 则范围不包括这个节点
	此时`startContainer`为开始节点的父结点
	`startOffset`为`startContainer`中, 开始节点的下一个索引
		即 开始节点为1, 则`startOffset`为2, 代表从开始节点之后开始范围
### 当开始节点为文本节点时
`startContainer`为该文本节点
`startOffset`为`startContainer`的一个索引
代表从该索引开始范围,包括该索引
## 结束 
### 当结束节点为元素节点时
+ 如果在一个节点之前结束, 则范围不包括这个节点
	此时`endContainer`指向结束节点的父结点
	`endOffset`为结束节点在`endContainer`的索引
+ 如果在一个节点之后结束, 则范围包括这个节点
	此时`endContainer`指向结束节点的父结点
	`endOffset`为结束节点在`endContainer`的索引+1
### 当结束节点为文本节点时
此时`endContainer`指向该文本节点
`endOffset`为该文本节点中的一个索引
代表从该索引结束, 包括该索引
