如果范围并没有选择文档的任何部分，则称为**折叠(collapsed)**
	即范围的开始与结束落在同一个位置
折叠范围有点类似文本框 : 
如果文本框中有文本，那么可以用鼠标选中以高亮显示全部文本
这时候，如果再单击鼠标，则选区会被移除，光标会落在某两个字符中间
而在折叠范围时，位置会被设置为范围与文档交界的地方，可能是范围选区的开始处，也可能是结尾处

折叠范围可以使用 collapse()方法
# `collapse()` 方法
这个方法接收一个参数：布尔值，表示折叠到范围哪一端
true 表示折叠到起点，false 表示折叠到终点
要确定范围是否已经被折叠，可以检测范围的 `collapsed` 属性：
```js
range.collapse(true); // 折叠到起点
console.log(range.collapsed); // 输出 true
```
测试范围是否被折叠，能够帮助确定范围中的两个节点是否相邻
例如有以下 HTML 代码：
```js
<p id="p1">Paragraph 1</p><p id="p2">Paragraph 2</p>
```
如果事先并不知道标记的结构（比如自动生成的标记），则可以像下面这样创建一个范围：
```js
let p1 = document.getElementById("p1"),
	p2 = document.getElementById("p2"),
	range = document.createRange();
range.setStartAfter(p1);
	// 第一次设置起点, 没有设置终点
	// 终点会重置到与起点相同的位置
range.setEndBefore(p2);
console.log(range.collapsed); // true
```
