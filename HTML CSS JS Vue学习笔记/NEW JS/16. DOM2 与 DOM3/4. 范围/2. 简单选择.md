 通过范围选择文档中某个部分最简单的方式，就是使用 `selectNode()` 或 `selectNodeContents()` 方法
 这两个方法都接收一个节点作为参数，并将该节点的信息添加到调用它的范围
 但选择的范围不同
+ `selectNode()`方法 : 
	`selectNode()`方法==选择整个节点，包括其后代节点==(包括传入节点)
+ `selectNodeContents()`方法 : 
	`selectNodeContents()` ==只选择节点的后代==(不包括传入节点)
---
```js
<!DOCTYPE html>
<html>
	<body>
		<p id="p1"><b>Hello</b> world!</p>
	</body>
</html>
```
以下 JavaScript 代码可以访问并创建相应的范围：
```js
let range1 = document.createRange(),
range2 = document.createRange(),
p1 = document.getElementById("p1");
range1.selectNode(p1);
// range1 包含\<p>元素及其所有后代
range2.selectNodeContents(p1);
// range2 包含\<b>元素、文本节点"Hello"和文本节点" world!"
```

# 设置范围之后Range属性的变化
## 调用 `selectNode()` 时
`startContainer`、`endContainer` 和 `commonAncestorContainer` 都等于==传入节点的父节点==
`startOffset` 属性等于==传入节点在其父节点 `childNodes` 集合中的索引==
```js
<!DOCTYPE html>
<html>
	<body>
		<p id="p1"><b>Hello</b> world!</p>
	</body>
</html>

let range1 = document.createRange()
p1 = document.getElementById("p1");
range1.selectNode(p1);
```
在这个例子中
`startContainer`、`endContainer` 和 `commonAncestorContainer` 为`body`元素
	选取中第一个子节点为`<p>`, 它的父结点为`<body>`, 所以`startContainer`为`<body>`
	选取中最后一个子节点为`<p>`, 它的父结点为`<body>`, 所以`endContainer`为`<body>`
	`startContainer` 与 `endContainer`的后代仅有`<p>`
`startOffset` 属性为1
因为传入的节点`p1`在`body`元素的`childNodes`中索引为1
	因为元素`p1`前面还有一个空文本节点
`endOffset` 属性为`startOffset`属性值+1
## 调用`selectNodeContents()`时
```js
<!DOCTYPE html>
<html>
	<body>
		<p id="p1"><b>Hello</b> world!</p>
	</body>
</html>

let range2 = document.createRange()
p1 = document.getElementById("p1");
range2.selectNodeContents(p1);
```
`startContainer` , `endContainer` 和 `commonAncestorContainer` 属性就是传入的节点
	在这个例子中是\<p>元素
`startOffset` 属性始终为 0，因为范围从传入节点的第一个子节点开始
 `endOffset` 等于传入节点的子节点数量（`node.childNodes.length`）
	在这个例子中等于 2

# 设置范围之后
在对Range对象调用完`selectRange()` 与 `selectNodecontents()`之后
还可以在范围上调用相应的方法，实现对范围中选区的更精细控制
相应的方法如下(也可以直接修改对应的属性值) : 
+ 设置起点
	+ `setStartBefore(refNode)`
		把范围的起点设置到 `refNode` 之前
		从而让 `refNode` 成为选区的第一个子节点
		`startContainer` 属性被设置为 `refNode.parentNode`
		`startOffset` 属性被设置为 `refNode` 在其父节点 `childNodes` 集合中的索引
	+ `setStartAfter(refNode)`
		把范围的起点设置到 `refNode` 之后
		从而将 `refNode` 排除在选区之外, 让其下一个同胞节点成为选区的第一个子节点
		`startContainer` 属性被设置为`refNode.parentNode`
		`startOffset` 属性被设置为 `refNode` 在其父节点 `childNodes` 集合中的索引加 1
	+ 注意 : 一个Range对象在创建时, 起点与终点位置相同
		==在没有设置终点, 直接设置起点时==
		==终点会重置到与起点位置相同的位置==
		会导致范围的折叠
		转到[[6. 范围折叠|范围折叠]]
+ 设置终点
	+ `setEndBefore(refNode)`
		把范围的终点设置到 `refNode` 之前
		从而将 `refNode` 排除在选区之外, 	让其上一个同胞节点成为选区的最后一个子节点
		`endContainer` 属性被设置为 `refNode.parentNode`
		`endOffset` 属性被设置为 `refNode` 在其父节点 `childNodes` 集合中的索引
	+ `setEndAfter(refNode)`
		把范围的终点设置到 `refNode` 之后
		从而让 `refNode` 成为选区的最后一个子节点
		`endContainer` 属性被设置为 `refNode.parentNode`
		`endOffset` 属性被设置为 `refNode` 在其父节点 `childNodes` 集合中的索引加 1
