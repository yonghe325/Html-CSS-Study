# `compareBoundaryPoints()`方法 -> 比较范围位置
该方法确定范围之间==是否存在公共的边界==(起点或终点)
这个方法接收两个参数：
+ 要比较的范围
+ 一个常量值 : 表示比较的方式
	这个常量参数包括: 
	+ `Range.START_TO_START` : 比较两个范围的起点(掩码为0)
	+ `Range.START_TO_END` : 比较第一个范围的起点和第二个范围的终点(掩码为1)
	+ `Range.END_TO_END` : 比较两个范围的终点(掩码为2)
	+ `Range.END_TO_START` : 比较第一个范围的终点和第二个范围的起点(掩码为3)

该方法返回一个数值, 代表两个范围边界的位置关系 : 
1. `-1` : 表示第一个范围的边界在第二个范围的边界 ==之前==
2. `0` : 表示第一个范围的边界在第二个范围的边界 ==重合==
3. `1` : 表示第一个范围的边界在第二个范围的边界 ==之后==

```js
let range1 = document.createRange();
let range2 = document.createRange();

let p1 = document.getElementById("p1");

range1.selectNodeContents(p1);
range2.selectNodeContents(p1);

range2.setEndBefore(p1.lastChild);

console.log(range1.compareBoundaryPoints(Range.START_TO_START, range2)); // 0
console.log(range1.compareBoundaryPoints(Range.END_TO_END, range2)); // 1
```
