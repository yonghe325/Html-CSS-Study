期约实例的方法是连接外部同步代码与内部异步代码之间的桥梁
这些方法可以访问异步操作返回的数据
处理期约成功和失败的结果，连续对期约求值，或者添加只有期约进入终止状态时才会执行的代码
	即在期约转换状态之后, 应该执行的代码
	
---
在 ECMAScript 暴露的异步结构中
	`Promise`类型就是一种异步结构
简单来说, 任何拥有 `then` 方法的对象就是 Thenable
	`Promise`类型, 就是`Thenable`的一种特殊实现
下面的例子展示了实现这一接口的最简单的类
```js
class MyThenable {
	then() {}
}
```
ECMAScript 的 Promise 类型实现了 `Thenable` 接口
