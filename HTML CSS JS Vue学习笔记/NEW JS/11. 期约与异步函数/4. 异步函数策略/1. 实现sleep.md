JavaScript由于为单线程, 所以不允许有sleep这种的 **同步阻塞主线程**
	否则会直接卡死主线程
ES8之后, 有了异步函数
就可以实现 **异步阻塞** 的sleep方法
```js
async function sleep(delay) {
	return new Promise((resolve) => setTimeout(resolve, delay));
}
async function foo() {
	const t0 = Date.now();
	await sleep(1500); // 暂停约 1500 毫秒
	console.log(Date.now() - t0);
}
foo();
// 1502
```
