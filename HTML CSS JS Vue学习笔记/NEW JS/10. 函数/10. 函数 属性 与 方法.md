在ECMAScript中, 函数是对象
所以函数也会有属性与方法
每个函数都有两个属性：`length` 和 `prototype`
# `length`属性
`length` 属性保存函数==定义==的命名参数的个数
```js
function sayName(name) {
	console.log(name);
}
function sum(num1, num2) {
	return num1 + num2;
}
function sayHi() {
	console.log("hi");
}
console.log(sayName.length); // 1
console.log(sum.length); // 2
console.log(sayHi.length); // 0
```
