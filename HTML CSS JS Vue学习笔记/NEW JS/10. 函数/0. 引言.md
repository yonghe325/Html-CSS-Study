在ECMAScript中
函数实际上是一个对象
每个函数都是`Function`类型的实例
所以与其他引用类型一样,Function也有属性和方法


因为函数式对象, 所以函数名就是指向函数对象的指针
而且不一定与函数本身紧密绑定

# 函数的声明
以不同方式声明函数
会有"微妙但重要的差别"
都是都能够创建函数
## 直接声明函数
```js
function sum (num1, num2) {
	return num1 + num2;
}
```
## 函数表达式声明
```js
let sum = function(num1, num2) {
	return num1 + num2;
};
```
与直接声明函数几乎等价
## 箭头函数
更多关于箭头函数, 转到[[1. 箭头函数|箭头函数]]
```js
let sum = (num1, num2) => {
	return num1 + num2;
};
```
## Function构造函数
接受多个字符串参数, 最后一个参数为函数体, 其它参数均为该函数的参数
```js
let sum = new Function("num1", "num2", "return num1 + num2"); // 不推荐
```
使用Function构造函数时
这段代码会被解释两次
第一次被当做常规的ECMAScript代码
第二次是及时传给构造函数的参数
这会消耗更多性能
